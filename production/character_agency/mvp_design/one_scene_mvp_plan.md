# One-Scene MVP Plan

本计划只设计实验，不生成正文。

## 对照组

A = baseline scene brief

B = Human Texture + Work Voice

C = Character Agency + Human Texture + minimal Work Voice

## 实验目标

- C 是否比 B 更有角色主动感。
- C 是否减少作者推着人物走。
- C 是否没有变成主角永远正确。
- C 是否保留网文推进。
- C 是否没有增加解释性心理描写。
- C 是否有更清晰的后果继承。

## 输入建议

优先选一个已经在 Human Texture 中验证过但仍可增强行动主权的片段，例如：

- C4 柳青砚关系节点。
- C3 饭堂 / 矿洞信息露出节点。

## 输出物

- `scene_agency_packet`
- `consequence_ledger` 初始条目
- `relationship_debt_ledger` 初始条目
- A/B/C brief 对照
- reviewer gate 评分

## 不做

- 不生成完整正文。
- 不运行 Hermes / DeepSeek。
- 不修改 skill-pack。
- 不读取 raw corpus。
