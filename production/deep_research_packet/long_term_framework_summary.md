# 长期框架摘要

这是当前项目判断，不是外部研究结论。

## 主线 A：当前生成链路增强

主线 A 不推翻当前 skill-pack 链路，而是在规划者、写作者、审核者、润色者之间逐层补强正文质量。

1. 人物质感（Human Texture）：解决人物不像功能件的问题。
2. 作品声音 / 作者站位（Work Voice）：解决正文里“谁在讲故事、讲述者站在哪里”的问题。
3. 角色主动感 / 行动主权（Character Agency）：解决人物是否自己推动局面的问题。
4. 活人浅痕 / 活人漏痕（Live Leakage）：解决人物是否有轻微、不刻意的活人痕迹。
5. 读者代入感 / 读者解压（Reader Immersion）：解决读者是否有参与判断、脑补和期待空间。

项目当前判断是：人物质感（Human Texture）更像基础防烂层，作品声音 / 作者站位（Work Voice）是叙述者位置层，角色主动感 / 行动主权（Character Agency）可能是后续关键转折层。

## 主线 B：角色驱动叙事引擎（Agentic Narrative Engine）

角色驱动叙事引擎（Agentic Narrative Engine）是长线研究，不立即替换当前 production pipeline。

它的候选组件包括：

- IP 宇宙（IP Universe）：定义题材承诺、世界资产和核心矛盾。
- 世界状态（World State）：保存事实、资源、制度、位置和事件后果。
- 角色 agents（Character Agents）：让角色拥有目标、认知边界、误判、关系、资源和策略。
- 场景仿真（Scene Simulation）：在有边界的场景里运行角色互动。
- 事件日志（Event Log）：记录角色行动、冲突、信息流动和后果。
- 叙述渲染器（Narrative Renderer）：把事件日志渲染成正文。
- 审核门禁（Reviewer Gate）：审核类型承诺、节奏、爽点、人物主权和越界风险。

## 依赖判断

- 作品声音 / 作者站位（Work Voice）依赖人物质感（Human Texture）v0 进入 base，否则叙述合同容易悬空。
- 活人浅痕 / 活人漏痕（Live Leakage）依赖角色主动感 / 行动主权（Character Agency），否则容易变成人味表演。
- 读者代入感 / 读者解压（Reader Immersion）依赖作品声音 / 作者站位（Work Voice）和角色主动感 / 行动主权（Character Agency），否则容易变成作者解释读者应该怎么想。
- 角色驱动叙事引擎（Agentic Narrative Engine）可以并行研究，但不能直接替换当前链路。
