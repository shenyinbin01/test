# DCQG-C009: 内在代价

## Solves Writing Problem
主角很强但读者不觉得"爽"——因为太顺了。如何让强大主角有血肉感？

## Mechanism
1. **定义代价源**：主角的核心能力/身份/优势本身自带代价（而非外部敌人制造的困难）
2. **代价不是苦难**：代价体现在选择受限、情感矛盾、身份撕裂，而非单纯的"受伤""失去亲人"
3. **渐进加深**：随着剧情推进，代价的隐性面逐渐暴露
4. **不影响爽感**：代价是让爽更有重量，不是削弱爽

## How To Use
- **Planner**: Bible 中定义 internalized_pressure 字段
- **Writer**: 每章至少一处体现代价（选择困境、情感暗流、身份冲突）
- **Reviewer**: 检查代价是否被"遗忘"——连续 3 章未体现需标出
- **Polisher**: 强化代价瞬间的情绪张力

## Positive Abstract Example
主角拥有"时间回溯"能力，但每次回溯都会让体内某个器官老化三年。他越强、回溯越多、身体越接近崩溃。他在战斗中不得不计算"值不值得回溯"，这种计算本身就是一种内在压力。

## Negative Example
主角很强但每次打完说"好累啊"就算代价——这是口号式代价，无实质约束力。

## Boundary
- 代价必须是能力的同源产物，不能是无关的附加物
- 代价必须影响剧情选择，不能是"有但它不影响任何事"
- 适用于所有需要长线人物弧光的角色
