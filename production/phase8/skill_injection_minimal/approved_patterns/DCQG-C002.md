# DCQG-C002: 规则破局

## Solves Writing Problem
主角解决问题永远靠"我更强大""我有更高级的功法""我突然突破了"——力量碾压式解法导致审美疲劳。

## Mechanism
1. **先理解规则**：主角花费篇幅观察/推理/试探冲突所涉系统的规则
2. **找出漏洞/约束**：规则总有边界和例外，找到它
3. **利用而非破坏**：在规则框架内逆转局面，比直接打破规则更有智力爽感
4. **代价体现**：利用规则漏洞应付出相应的代价或风险

## How To Use
- **Writer**: 高潮解法优先考虑规则利用而非力量碾压
- **Planner**: chapter beat 中标注 conflict_resolution_type（force/rule/social）
- **Reviewer**: 检查是否连续 3 章以上都使用力量碾压式解法
- **Polisher**: 强化破局瞬间的"智性爽感"

## Positive Abstract Example
对手设下生死擂台，规则是"只许使用擂台上的兵器"。主角发现擂台护栏是导电金属——他不是用护栏当兵器（违反规则），而是用内力在护栏中制造静电场干扰对手的金属护甲。他在规则内找到了物理漏洞。

## Negative Example
主角说"规则是给弱者定的"，然后一拳打爆擂台——这是力量碾压，不是规则破局。

## Boundary
- 规则破局需要前期铺垫（读者需要理解规则才能享受破局）
- 每卷至少 2~3 个关键冲突使用规则破局
- 不适用于纯力量型章节（如战力展示、修炼突破）
