# 第一章 四十五点

青云宗的演武场上跪着五排人。新进弟子入门测试——测经络积分。

前面的四个人已经测完了。八十九。七十六。九十一。八十二。测试官面无表情地念着数字，每念一个，人群里就有一阵低低的骚动。

"下一个。"

苏衍站起来的时候，旁边的几个人看了他一眼。他太瘦了，衣服洗得发白，在一众穿着崭新道袍的弟子中间显得很突兀。他走到测试石前，把手按上去。

石头亮了一下。

测试官低头看了看读数，声音比刚才轻了几分：“四十五点。”

全场安静了大约三次呼吸。

然后苏衍听到了笑声。不是那种哈哈大笑，是那种从牙缝里挤出来的、压抑着的嗤笑。站在他身边的胖少年——就是那个九十一分的——用肩膀撞了撞旁边的人：“四十五？这也能进青云宗？”

“杂役弟子。”测试官在竹简上划了一下，“下一个。”

苏衍把手从测试石上抽回来。他的手指有点发麻，不是因为测试石，是因为他刚才看到了一件不该看到的事——测试官手指上的经络积分，在虚空中微微颤抖了一下。像水面上的倒影被石子击中，泛起了一圈金色的涟漪。

他眨了眨眼。涟漪消失了。

他被领到了宗门最东边的杂役宿舍。说是宿舍，其实是三间漏雨的石头屋子。领路的杂役师兄一路没跟他说话，只走到门口时丢了一句：“饭堂在北边，往后院走。别去前山，那里不是你们待的地方。”

苏衍嗯了一声。

他从杂役宿舍出来的时候天色已经暗了。前山的主殿方向传来经脉运转的声音——那些积分高的新弟子正在主殿里演练基础功法。苏衍站在后山的小路上，看主殿的屋顶在夜色中泛着微光。

然后他又看到了。

那些人在运转功法的时候，经络周围有一层淡淡的金色光晕在流失——像水从裂口的桶里渗出，悄无声息地散入虚空。别人看不见。但他能看见。

而且不止能看见。

苏衍下意识地朝那股溢出伸出了手。没有碰到任何东西，但他感觉到指尖一凉——一道极细的金线从虚空中被拉了出来，顺着他的指尖钻进了经络。

身体深处传来一声微响，像琴弦被拨动。

他低头看了看自己的掌心。经络积分从四十五变成了四十六。

苏衍在黑暗中站了很久。

那天夜里他没有睡觉。他坐在杂役宿舍的门槛上，听着前山主殿的方向。每隔大约半个时辰就有人运转一圈功法，然后就有金色的光晕从那边飘过来——微弱的、散碎的、别人不要的——像夜风里飘落的花粉。

他一次次伸手去接。五点。八点。十二点。

当他的积分涨到五十二的时候，一阵莫名的烦躁突然涌了上来。不是他的情绪。是那个胖少年白天运转功法时的竞争欲——满脑子都是"我要踩在所有人头上"的念头——现在这些念头像碎片一样在他的意识里浮沉。

苏衍猛地睁开眼。他的手指在发抖。

回收溢出不是没有代价的。他在接收别人的积分，也在接收那些积分的"杂质"——修行者运转功法时的情绪残留。贪婪。恐惧。愤怒。随着积分增多，这些不属于他的情绪也会越来越多。

他攥紧了拳头把那股烦躁压下去，然后抬头看了一眼前山的屋顶。

明天是实战测试。他的对手是那个七十六分的。

第二天，演武场。

柳青砚的剑比苏衍快太多了。第一剑就劈在苏衍的肩膀上，第二剑扫过他的大腿。围观的弟子们一片哄笑——四十五分的废物果然撑不过三回合。

但苏衍没有在躲。他在看。

柳青砚每出一剑，经络里的积分就会有一小部分从剑尖溢出。那股金色的光晕流入虚空，然后被苏衍的经络牵引着回收。他被击中一次，积分反而涨了一分。被击中两次，涨了两分。

柳青砚打到第三回合的时候终于发现了不对劲。苏衍的气息没有减弱，反而在增强。

“你——”柳青砚的剑停了一瞬。

苏衍等的就是这一瞬。他已经看清楚了——柳青砚的功法运转有一个节奏，每次全力出剑之后，经络会有大约两次呼吸的虚浮期。在那个瞬间，功法的防御最弱。

柳青砚第四剑劈来的时候，苏衍没有躲。他用自己的经络硬接了那一剑，同时在接触的瞬间把从柳青砚身上回收来的积分顺着剑身反震回去。

不是比力量。是在规则内找到漏洞。

柳青砚的剑被震飞了。他退了半步，低头看着自己空了的手，脸上的讥讽变成了错愕。

全场安静。

测试官慢慢地站了起来。他的目光从柳青砚身上移到苏衍身上，停了很久。

“四十五点。”他说，声音不大，但每一个字都清楚得不像是在耳语，“你的积分……涨了？”

苏衍没有回答他。他低头看了看自己的掌心——六十七点，还在涨。

然后一阵剧痛从后脑炸开。

不是身体的痛。是记忆。不属于他的记忆碎片像碎玻璃一样扎进意识——他看到了一个老人跪在古阵前，看到了满天金色的丝线像暴雨一样从虚空中坠落，看到了一双从黑暗中睁开的眼睛。

苏衍跪倒在演武场上。

测试官的声音从很远处传来：“叫医修——快叫医修——”

但苏衍听不见了。他的意识里只有那双眼，那双从虚空深处注视着他的眼。

以及一个声音。

那个声音说：你终于看见了。
