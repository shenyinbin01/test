# 技法资产卡：逐层揭示格局升级

- **技法类型**：payoff
- **解决的写作问题**：如何让读者在每章结尾感到"问题比想象的大"
- **适用题材**：悬疑、犯罪、都市、权谋
- **适用场景**：章尾、阶段性高潮
- **操作步骤**：
  1. 本章主角解决了一个小问题（拿到证据）
  2. 在解决问题后，通过一个信息（对话/物品/观察）揭示问题更大
  3. 主角的应对姿势从"解决"变为"面对"
  4. 章尾落在"新问题比旧问题大"的认知反转
- **预期阅读效果**：读者从"解决了"的满足感切换到"糟了"的焦虑感
- **不适用场景**：纯爽文不需要格局升级，只需要打脸
- **常见失败方式**：格局升级得太快，读者跟不上；格局升级出现在开头而非结尾，浪费了悬念
- **原作污染风险**：low
- **禁止复制内容**：具体金额数字、集团总部设定
- **适合注入哪个 Skill**：planner, reviewer
- **证据来源**：追踪者 第3章 — 拿到证据后老周揭示资金流向集团总部
- **审核状态**：candidate
- **版本**：v1
