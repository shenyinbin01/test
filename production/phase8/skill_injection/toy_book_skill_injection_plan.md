# Skill 反哺方案：追踪者

## Planner 可调用技法

- `pacing_三章节奏模板` (approved) — 前三章节奏控制
- `opening_hook_信息缺口` (candidate) — 开篇悬念设计

## Writer 可调用技法

- `pacing_三章节奏模板` (approved) — 可安全执行

> **安全约束**：Writer 禁止接触原文、禁止接触 source evidence、禁止接触 candidate 技法。

## Reviewer 可调用技法

- `payoff_逐层揭示` (candidate) — 检查章尾是否有格局升级
- `pacing_三章节奏模板` (approved) — 检查节奏是否符合模板

## Polisher 可调用技法

- （暂无）

## 每个 Skill 的禁止输入

- Planner：禁止接触原作人物名和具体情节
- Writer：禁止接触原文、禁止接触 source evidence、禁止接触 candidate 技法
- Reviewer：无禁止
- Polisher：无禁止

## 每次任务最大注入技法数量

3

## 注入触发条件

- 新书启动时：注入 opening_hook 和 pacing 模板
- 章尾审稿时：注入 payoff 检查
- 前三章完成时：注入三章节奏模板检查

## 使用日志要求

每次技法调用必须记录：使用的技法ID、注入的章节、效果评估。

## Reviewer 验收方式

- 检查技法是否被正确使用
- 检查是否有原作污染（原文表达、人物名、设定名）
- 如果发现污染，标记为 contamination_risk 并退回

## 原作污染检查方式

- 随机抽样正文段落与 source evidence 对比
- 检查专有名词是否被去原作化
- 检查桥段是否保留了原作的独特设定
