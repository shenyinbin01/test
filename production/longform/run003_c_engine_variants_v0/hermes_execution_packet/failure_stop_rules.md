# Failure Stop Rules

Create `stop_report.md` and stop if any variant triggers:

- Prose contains engineering fields.
- Prose contains Orchestrator notes.
- Prose contains event_log.
- Any state_delta is marked accepted.
- Any ledger is marked official.
- Raw corpus is used or saved.
- A specific author is imitated or used as target.
- Variant inputs diverge from unified source.
- C3 output is harder and more process-like than C0 by initial checklist.
- LLM reviewer chooses a winner.
- LLM reviewer replaces human blind evaluation.
- Hermes rewrites input contracts during execution.

Stop report must include variant, chapter, evidence excerpt, violated rule, and recommended return layer.
