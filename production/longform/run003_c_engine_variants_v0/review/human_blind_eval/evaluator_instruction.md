# Evaluator Instruction

Read each anonymized pair without knowing which system produced it.

Prefer the version that feels more like fiction and less like process. Do not reward clearer explanation if it makes the scene feel managed. Do not reward ornate language if the choice, scene heat, and relationship consequence are weaker.

Answer the sheet in plain terms:

1. Which version makes you want to continue?
2. Which version feels less AI?
3. Which version makes the protagonist feel like he can only choose this flawed option?
4. Which version has more scene heat?
5. Which version leaves better aftertaste and room to infer?

You are not judging engineering completeness.
