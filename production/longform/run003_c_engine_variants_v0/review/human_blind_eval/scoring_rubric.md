# Human Scoring Rubric

Score 1-5 when needed.

| Dimension | 1 | 3 | 5 |
|---|---|---|---|
| reader_pull | no desire to continue | mildly curious | strongly wants next chapter |
| anti_ai_surface | process-like | mixed | fiction-like |
| character_alive | functional piece | partly pressured | feels like person under constraint |
| choice_inevitability | plot-required | understandable | flawed but emotionally inevitable |
| scene_heat | information exchange | some friction | tactic, counter-tactic, turn, cost |
| subtext | all explained | some inference | strong unsaid pressure |
| anti_explanation | explains design | occasional explanation | trusts reader |
| technique_invisibility | machinery visible | faint traces | no visible engine |
| relationship_pressure | labeled only | somewhat felt | changes behavior/future options |
| longform_expandability | closed | has some next seed | strong next-friction seed |
