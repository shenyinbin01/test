# C0 Private Engine Policy

C0 may use current C anchor planning internally, but all operational artifacts remain private.

## Private

- C-style agency notes
- Orchestrator notes
- spotlight/focus notes
- draft event summaries
- reviewer checklist

## Public To Renderer

- natural-language scene brief
- safe continuity summary
- focal stance
- forbidden terms

## Hard Stop

Stop if the prose contains an engineering label or if the renderer receives private operational fields raw.
