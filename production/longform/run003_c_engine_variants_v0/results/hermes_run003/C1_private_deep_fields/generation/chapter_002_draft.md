# 第二章

第二关的考场比第一关小得多。

一间没有窗的房间，四张工作台围成菱形，每张台面上放着一台笔记本电脑和一份密封的纸质材料。墙上的挂钟秒针一下一下跳，声音很响，像是在给什么倒计时。

沈主任坐在角落的观察席，面前摊着评分表，笔帽还没摘。他说："第二关，策略协同。规则很简单——你们四个人拿到的数据不完整，所有人必须交换信息才能完成拼图。限时四十分钟。"

林延撕开材料封条。他的那部分是一组供应链节点的断裂数据，缺了上游的货源分布和下游的渠道权重。他抬头看了一眼对面的苏棠，苏棠正在看自己的材料，表情像在读一份普通的月度报表。

"分配一下，"赵锐的声音从右边传过来，"各自报缺口，十分钟内对出重合区。"

赵锐说话的时候眼睛扫了一圈，停在林延身上。林延没看他，把材料摊开，报了自己缺的部分。苏棠第三个报，声音平稳，"下游权重在我这里。"

这意味着林延必须主动跟苏棠要数据。

"下游权重，"林延说，"能发我一份吗？"

"已经发你了，"苏棠说，没抬头，"共享文件夹里。"

林延点开文件夹，数据确实在里面。格式规整，标注清晰，是苏棠一贯的标准。但没有多余的话。没有"你看看这组数据有没有问题"，没有"你觉得上游怎么推"。

就是数据。干干净净的数据。

林延开始拼图。手指在触摸板上滑动，屏幕的蓝光映在他脸上。拼到第十五分钟时他卡住了——上游货源分布的缺口比他预想的大，他自己的数据包里有一组数字对不上赵锐报出的总量。

"上游总数不对，"林延说，"你们谁有完整的货源分布？"

赵锐敲了两下键盘，"我有一部分。不过——"他顿了顿，语气忽然变轻了，像是在聊一件无关紧要的事，"这种需要交叉验证的数据，有人提前做功课的话会快很多。"

说这话时赵锐正盯着自己的屏幕，但他眼角的余光偏了一下。偏的方向是林延。

林延的手指在触摸板上停了一瞬。不到一秒。他继续拖动数据，像什么都没听见。

但苏棠听见了。或者说，她看见了林延手上那一瞬的停顿。

她没说话。继续整理自己的那部分拼图，键盘声均匀，像缝纫机的针脚。

时间走到二十五分钟。林延的拼图缺最后五个节点的映射关系，这五个节点不在一开始分配的任何一份材料里——它们散落在四个人的数据边缘，需要把所有人报出的数字重新拆解重组才能定位。

"最后五个节点需要倒推，"林延说，"每个人报自己数据包里的异常值。"

赵锐先报了。然后是另一个候选人。然后是苏棠。

林延把四个人的异常值拉进表格，开始跑匹配。屏幕上的数字跳了几下，最后一行卡在百分之九十三——缺一个映射。

"谁的没漏出来？"赵锐问。

沉默了几秒。苏棠说，"我的异常值里有一个可能是交叉项，刚才没识别。"

"现在补？"

"来不及了，"林延看着时钟，"二十八分钟。直接推。"

他做了一个快速判断——用赵锐报的一个相近数值替代苏棠的模糊项，强行补上映射链。拼图逻辑在技术上是通的，但偏差百分之二。

时间到。沈主任站起来收评分表，看了一眼四台屏幕上的结果，没说好坏，只说了句"第三关安排明天通知。"

走出考场时走廊里的空气像刚被抽过真空。林延的衬衫后背湿了一块，粘在皮肤上，空调一吹发凉。苏棠走在前面三步远的位置，步子不快，但也没有慢下来等他的意思。

"苏棠。"林延叫了一声。

她停下来，回头。

走廊里就他们两个。灯管嗡嗡响。林延想说点什么——关于刚才那一秒的停顿，关于为什么他没有解释，关于赵锐那句"提前做功课"。

但他没说。因为任何解释都会牵出老周的消息，而那条消息他到现在也没告诉过她。

"刚才那个替代值，"林延最后说，"偏差应该能控制在可接受的范围内。"

苏棠看了他两秒。然后说，"偏差不在你的映射链上。"

她转身继续走，帆布鞋踩在水磨石地面上，声音比昨天轻。像是走得更远了，连脚步声都在往回收。

林延站在走廊里。苏棠刚才那句话的意思他听懂了——偏差不在技术上。

但他不知道偏差在哪里。

不对。他知道。他只是没办法开口。

走廊尽头的窗外是下午四点的太阳，光线斜着照进来，在墙上切出一个锐角。林延的手机在口袋里震。不是老周。是日程提醒：明日第三关，最终评估。

他把提醒划掉，屏幕灭掉。

从什么时候开始的？从他说"行"但没说"我还有一条消息"的时候。从老周办公室出来，他选择一个人走下楼梯的时候。从苏棠回了一个"好"，干净得不像她的时候。

这些都不是大事。每一件单独拎出来都小到可以忽略。但把它们摞在一起，就变成了苏棠今天看他的那个眼神——不是生气，不是失望。

是评估。

她在评估他。像一个合作者在评估另一个合作者的可信区间。

这个眼神比任何质问都让林延难受。因为如果是质问，他可以解释。但评估不需要回答。评估只需要数据。而苏棠已经有了足够的数据。

林延往电梯走。按了下行键，指示灯从五跳下来，一格一格地落。

电梯门开了，里面站着赵锐。

赵锐看见他，没动身子，只抬了抬下巴，算打招呼。林延走进去，站在他对角线的位置。电梯下行，数字跳变。

"你第一关做得不错，"赵锐说，声音很随意，"有人跟我提过。"

林延没回头。"谁？"

"不重要。"赵锐走出去，电梯门在他身后关上。

不重要。

林延一个人在电梯里，看着不锈钢门板上自己变形的倒影。不重要——但赵锐知道有人给林延递过消息。这件事本身就是一个消息。

电梯停在一楼。门开了，外面是大厅，人不多。林延走出去，经过公告栏时停了一下。栏里贴着上次考核的结果公示，他的名字在中间，苏棠的名字在上面一行。

他盯着那两行名字看了几秒，然后推门出去。

外面还在下雨，比昨天小。毛毛细雨落在脸上没什么感觉，但时间长了会湿透。

林延没戴帽子。他沿着人行道往地铁站走，头发慢慢贴在了额头上。经过昨天那个水洼时他看了一眼——路灯的光还是碎在上面的，但水洼比昨天浅了。

快干了。但不是今天。
