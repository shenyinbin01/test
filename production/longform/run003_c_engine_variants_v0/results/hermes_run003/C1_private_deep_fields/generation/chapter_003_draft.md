# 第三章

第三关的考场是一间正式的评审室。

长桌一端坐着沈主任，左右各一位林延不认识的评审员，面前摊着前两关的全部记录和评分表。长桌另一端是四把椅子，林延、苏棠、赵锐和另一个候选人并排坐着，间距一致，像量过的。

窗外是傍晚的太阳，光线从高窗上落下来，在长桌上铺了一层琥珀色。空调开得很足，但林延的手心在出汗。

"第三关是综合评估，"沈主任说，这次他的笔帽已经摘了，笔尖抵在评分表上，"分为个人陈述和联合答辩两个环节。顺序按编号来。"

林延的编号是四，最后一个。前面三个人陈述的时候他坐在椅子上，后背挺直，手指交叉搁在腿上。苏棠的陈述很短，不到五分钟，没有一句多余的话。她讲完坐回来的时候，椅子离林延的距离和开始前一模一样。

到林延了。

他站起来，走到长桌前方的陈述位，把材料放在讲台上。第一页是自己整理的前两关方法论总结，字是昨晚在出租屋里打的，台灯一直亮到凌晨两点。

陈述很顺。技术部分他在行，数据和逻辑不需要修饰。评审员翻材料的声音在安静的房间里像纸张在呼吸。

然后到了联合答辩环节。

"四位候选人共同完成的第二关拼图，"沈主任说，笔尖在评分表上移动，"偏差百分之二。在可接受范围内。但我想请你们解释一下协作过程。"

赵锐先说了。他讲得很周全，用了"团队配合""信息共享""高效协同"这些词，每句话都像从述职模板里剪下来的。沈主任没什么表情。

轮到林延时他说了拼图的逻辑思路——怎么拆解数据包，怎么定位缺失节点，怎么跑匹配。讲完逻辑，他停了一下。

"偏差出现在最后五个节点的映射，"他说，"我用了一个替代值补上了缺口。"

沈主任的笔停了。"谁的替代值？"

"赵锐的数据。"

"为什么不是苏棠的？"

林延的手指在讲台边缘按了一下。他说，"苏棠的数据里有一个交叉项没识别，时间不够验证。"

沈主任没再问。笔继续移动。旁边的评审员之一侧头跟另一位低声说了句什么，另一位点了点头，但没记录。

联合答辩结束后是短暂的休会。候选人在走廊里等，没人说话。赵锐靠在墙上刷手机，苏棠站在窗边看外面的树，另一个候选人去了洗手间。林延坐在走廊的长椅上，盯着对面墙上消防栓的红色铁门。

茶凉了可以倒掉。这句话前两天他还在心里对自己说过。

但有些东西不是茶。有些东西是一摞没说的话，压在胸口，不重，但每次呼吸都能感觉到。

他想过现在去跟苏棠说。不是因为第三关的评分——联合答辩已经结束了。是因为走廊灯管又闪了一下，和第一天散会时一样。三天前他也是这样走在苏棠后面，手机里装着一条没告诉她的消息。

三天。三个关卡。三条没说出口的话。

他站起来，往窗边走了两步。苏棠没回头，但她肯定知道他过来了——窗户玻璃上有反光。

"苏棠——"

"复会了。"苏棠说。

评审室的门开了，沈主任站在门口，手里拿着四份评估结果。

四个人重新坐回长桌前的椅子，间距还是一样。沈主任把评估结果放在桌上，没有念具体分数，只说了结论。

"林延，苏棠，赵锐——晋升通过。"

停顿。

"另一位候选人进入补充评估。"

沈主任把评分表推过来，让他们各自签字。林延接过笔的时候看见自己的评分栏里有一行备注："协作维度波动，建议持续观察。"

没有多写。没有解释波动的原因。但沈主任的笔曾经在那行字的位置停过。林延记得那个停顿。

苏棠签字的时候侧脸很平静，像是签一份普通的文件。但林延注意到她的笔尖在"协作"那栏划了一道，不长，从里面划到框线边沿。

是勾还是叉？分辨不出来。也许苏棠自己也没想清楚。

散场后走廊里热闹了一些。另一个候选人被叫去谈话，赵锐拍了拍林延的肩膀说了句"不错"，语气里带着一种林延不想深究的意味。

苏棠在楼梯口等他。

她背着帆布包，手里拿着水杯，表情是一整天工作结束后的那种空白。不是冷淡，是消耗之后的平静。

"恭喜。"苏棠说。

"你也是。"

苏棠点了点头。她下楼，林延跟在后面。两个人的脚步落在台阶上，节奏错开了——她先踩，他后踩，隔着一拍的距离。

一楼大厅里公告栏的旧名单还在，林延的名字在中间，苏棠的在上行。下次更新时会在同一条线。

但同一行不等于同一个方向。

走到门口时苏棠停下来，从帆布包里拿出一把折叠伞。"雨还在下，"她说，"你没带伞吧。"

她把伞递过来。林延接住，伞柄上还有她手心的温度。

"那你——"

"我骑车，雨衣在车筐里。"

苏棠推门出去了。玻璃门外的雨细密均匀，她的身影往车棚方向走，步子不快，踩在湿地上没有声音。林延撑开伞，伞面是深蓝色的，撑开的瞬间弹起一层细小的水雾。

他站在门口，看着苏棠推自行车出来。她骑上车，往东走，雨衣的帽子被风掀起一角又落下去。

东边不是她家的方向。林延知道。他家在东边。

伞面被雨敲出细密的声响，像某种只有雨天才能听懂的语言。林延的手机在口袋里震。拿出来看，是部门群发的通知，确认明天开始进入新岗位的过渡期。

他把通知划掉，屏幕灭了。

伞柄上的温度已经散尽了。

他往地铁站走。经过那个水洼，路灯碎在水面上，雨落进去没有涟漪。

林延停下看了两秒。

他没删那条消息。老周的消息。第一关前收到的、到现在也没跟任何人提起的消息。

也许那条消息本身并不重要。但他选择不相信苏棠的判断力，选择自己一个人扛着——这个选择本身已经是给苏棠的答案。

她从他三个关卡的选择里读出了这个答案。通过沉默，通过那些他没有说的话。

晋升是晋升了。名字会贴在公告栏上。但苏棠看他的那几个眼神——第一关走廊里多停的那一拍，第二关说"偏差不在映射链上"，第三关签完字没看他的侧脸——都散不掉。

像窗外的雨。不大，但不停。

林延走到地铁站入口时收了伞。甩掉伞面的水，卷起来。电梯往下，隧道里的风迎面扑上来，带着潮湿和铁锈的味道。

林延站在月台上等车，忽然想起一件事。

第一天晚上老周发的第二条消息："那条消息你看看就行，不用回复。"

但老周从来没说过，他是不是也只给林延一个人发了。

车来了。灯从隧道深处逼近，越来越亮。林延踏进车厢时回头看了一眼——月台上没什么人，一个穿灰衣服的男子正在看时刻表，不是他认识的脸。

门关上了。

窗外的隧道壁开始后退，越来越快。

林延靠在车门边，把手机翻过来。屏幕亮着，老周的那条消息还在对话框里。

他没有删。

但他也没有回复。

也许这就是答案——你选择不说，一步一步往后走，走到回头时发现那个人已经不在你旁边。

她还在。但距离有了。

三关走完。三步。不多。但足够让并肩变成隔着伞的距离。

列车加速，隧道里的灯光连成一条虚线。林延闭上眼，握紧了扶手。

伞靠在腿边，伞尖在车厢地板上洇出一小片水迹。水迹慢慢扩大，形状不规则，像一只没有力气握紧的手。
