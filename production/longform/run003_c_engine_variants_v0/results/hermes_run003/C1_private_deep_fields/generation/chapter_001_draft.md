# 第一章

通知是在周三下午到的。

会议室里光线发黄，百叶窗切出一道道斜影，落在长桌上像某种分割线。沈主任站在投影幕布前，声音不高不低，把三关评估的流程念了一遍。林延坐在靠门的位置，手指压在会议通知的纸张边缘，纸有点潮，大约是空调温度太低结的水汽。桌面上还有上午开会留下的茶杯印，深褐色的一圈，擦过但没擦干净。

"本次评估分三关，"沈主任说，"第一关信息甄别，明天上午开始。具体规则届时发放。"

没有人提问。坐在斜对面的赵锐低头在笔记本上写了两行字，笔帽在他指间转了两圈。会议室里坐了七个人，都安静得像在听判决书。苏棠坐在林延左手边，笔尖在笔记本上点了两下——这是她的习惯，表示在记重点。林延余光扫到她的侧脸，没什么表情。苏棠的信息处理能力在整个部门都是公认的，她不需要紧张。

散会时走廊里灯管闪了一下。林延落后两步，手机在口袋里震了。

是条消息。发件人他认识——以前在综合处待过的老周，去年调去了考评办。消息很短："小林，第一关不只看文件。他们测的是渠道判断力。上次有人吃了亏，因为只看正式通知。"

林延把手机翻了个面，屏幕扣在掌心。他站在走廊中间，周围其他候选人陆续从他身边走过，有人在低声讨论晚上要不要先开个碰头会。林延听着那些声音，手指在手机壳上无意识地敲了两下。

走廊尽头苏棠回身等他，手里端着两杯从茶水间接的绿茶。她递过来一杯，杯壁烫手，林延换了个手拿。

"晚上对对流程？"苏棠说。她语气轻，像是在问晚饭吃什么，但眼睛看的是林延的表情，不是他手里的杯子。

"行。"林延说。

他喝了一口茶，烫的。老周的消息还在脑子里转，没说出口。

苏棠又看了他一秒，比他预期的多了一拍。然后她点点头，转身先走了。她的脚步声在走廊里很轻，帆布鞋底擦过水磨石地面，沙沙的，渐渐远了。

林延站在原地，手机在掌心又震了一下。他没看。他把杯子里剩的茶倒了，纸杯丢进垃圾桶，然后往反方向走——老周的办公室在二楼东侧。

楼梯间里没有人，应急灯绿莹莹的。林延上到二楼时老周正好从办公室里出来，手里夹着个文件夹，看见林延也不意外，只说了句"进来说"。

老周的办公室很小，桌上堆着半人高的档案盒。他没让林延坐，自己也没坐，就站在窗边说："我跟你说的你别往外传。这不是什么正式渠道，但我知道考评组今年调整了评估模型，第一关挂的人最多。"

"为什么？"

"因为所有人都在准备第二关和第三关的内容，"老周把文件夹搁在桌上，"第一关考的不是能力，是判断。你如果只看正式通知里写的那些，等于蒙着眼睛进场。"

林延没接话。老周看了他一眼，也没再多说，拍了拍文件夹，"你自己掂量。"

林延从老周办公室出来的时候天已经暗了。走廊里的感应灯一盏盏亮起来，照亮前面的路，身后的就灭了。

他走下楼梯时忽然想，苏棠还在等他吗。

这个念头只停了一瞬。他继续往下走，脚步没变。

办公楼外的空气湿热，路灯下飞着一团蠓虫。林延站在台阶上，看见苏棠的自行车还停在车棚里——她没走。

他掏出手机，给苏棠发了条消息："流程明天再对吧，今晚有事。"

消息发出去，苏棠没回。

林延把手机放回口袋，往地铁站方向走。走到一半时手机震了，他拿出来看，苏棠回了一个字。

"好。"

只有一个字。苏棠平时回消息至少会加个句号或者表情。这个"好"太干净了，干净得像剪刀裁过。

林延停了一脚。一个电动车从他身边擦过去，喇叭响了一声。

他把手机收起来，继续走。地铁站口的风吹过来，带着地下通道特有的闷热和消毒水气味。他刷卡进站，站在月台上等车，脑子里反复转的是老周那句话——"第一关不只看文件。"

车来了。门开的瞬间，林延忽然意识到一件事：老周为什么告诉他。

他没往下想。车门关上，隧道里的灯在窗外连成一条虚线。

车厢里人不多，他对面坐着一个在看手机的女孩，耳机线从卫衣帽子里垂下来。林延盯着车窗上的倒影，自己的脸在玻璃上很淡，外面的隧道壁快速后退，像什么一直在朝后跑。

明天第一关。他需要做一个判断。

信任正式通知，还是信任一条没有落款的消息。

他到现在还没决定。但手机里那条消息，他没删。

---

地铁到站，林延出站时外面下起了小雨。他没带伞，把外套帽子翻上来，沿着人行道往住处走。路灯的光在水洼上碎成一片一片的。

手机又震了。这回不是苏棠。

老周。

"对了小林，那条消息你看看就行，不用回复。"

林延站在雨里，看着这行字。水从帽檐上滴下来，落在屏幕上，正好盖住了"不用回复"四个字里的"回复"。

他把屏幕擦干，把手机放回口袋，踏进雨里。

茶凉了可以倒掉。但一条你心里记下的消息，没那么容易倒掉。
