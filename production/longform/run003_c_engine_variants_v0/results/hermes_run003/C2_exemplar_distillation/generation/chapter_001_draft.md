# 第一章 入口

通知贴在公告栏的时候，走廊里只有林延一个人。

午休刚过，整层楼安静得像一张没写过字的纸。空调出风口嗡嗡响着，像某种不会停下来的警告。他站在那张A4纸前面，把第一行字读了两遍。三关晋升考核。第一关：信息甄别——三个工作日内提交策略方案，逾期视为放弃资格。

不是难题。他做过的方案比这多得多。每年考核季，办公区的打印机都要吐掉半箱纸，全是方案初稿和修改稿。他不缺经验，不缺技术，甚至不缺前两年积累下来的评估分数。

他缺的东西，他自己也说不清。

拿出手机拍了张照，准备转身回工位。屏幕亮了一下——不是拍照的回显，是一条陌生号码发来的消息。内容很短，短到只需要扫一眼就能记住：

「正式入口的第三条路径是冗余设计。真正的通道在附件标注的第五项。别走弯路。」

附件是一份非正式的流程说明。落款处没有署名，只有一串科室编码。赵锐的科室。

林延把屏幕按灭，站在原地没动。拇指压在电源键上，压到指节发白。走廊尽头的窗开着半扇，四月的风灌进来，吹得公告栏边角掀起又落下，啪啪地响。

他应该删掉它。

他没删。

走回工位的路上经过茶水间，苏棠正靠在窗边等他。她穿了件深蓝衬衫，袖口卷到手肘，小臂上沾了一道水笔印子，大概是上午改方案蹭上去的。她手里捏着保温杯，枸杞泡得发胀，红色的果子在透明的杯壁里浮浮沉沉。她看见他就站直了身子，像是等了有一会儿。

“公告看了？”苏棠问。

“刚看完。”

她往他这边走近半步，压低了声音。茶水间里只有他们两个人，但她还是把声音压得很低，像在说一件需要藏进喉咙里的事。“我听说了件事。正式通知里的材料清单，第三条路径的索引是故意放的烟雾弹。如果按那个走，方案一开头就偏了。评审组就是要看谁能识破。”

林延握着手机的手紧了紧。手机在裤兜里贴着他的大腿，屏幕的余温好像还没散尽。

苏棠的信息来源不是公开渠道。她没说从哪里听来的，这就意味着那条信息本身也需要她用什么东西来交换。她说出来的代价，是信任——信任他不会追问来源，不会在后续考核里把这条信息当作她的把柄，不会让她为说出这句话而后悔。

他应该说：我也收到了一条。

陌生号码。科室编码。第五项路径。两个人把信息拼在一起，至少能互相验证，至少能让彼此知道对方手里有什么牌。

但他没说。

“你打算怎么走？”苏棠问。

林延把目光从她脸上移开，落在窗外那棵银杏树上。树是新栽的，枝干细得撑不住风，叶子倒很密。“我先看看正式通知，”他说。“把材料过一遍再定。”

苏棠的表情变了一下。很轻微，轻微到如果不是认识她三年以上根本看不出来——嘴角往下压了不到一毫米，然后恢复了正常。她把保温杯搁在茶水间的台面上，杯底磕在瓷砖上，发出一声脆响。

“行。”她说了一个字。

她端起杯子走了。经过他身边的时候没有侧头，没有停顿。保温杯里的热气从杯口溢出来，在空气里散成一小团白雾，很快就被走廊里的风搅散了。

林延看着那团白雾消散，想起三个月前的那件事。

那时候他们还在同一个项目组。苏棠为了帮他争取一个外派名额，动用了她导师在总部的关系。事后导师在部门会上意有所指地提过一句——说某些年轻人不要总想着借别人的信任铺路，路铺得再好，走的人也得自己会走。

那句话不是对他说的。

但会议室里所有人都知道说的是谁。苏棠坐在最后一排，脸白得像桌上的打印纸，从头到尾没有抬头。

会后他去找她，她说没事。他说谢谢。她说不用谢。

然后他们之间就多了一样东西——一种说不清道不明的、悬在两个人中间的债务。不是钱的债，是信任的债。她借给他一次，他就欠她一次开口的权利。她再帮他一次，他就更加开不了口。

下午四点，林延坐到工位上打开文档。光标在空白页面上闪。

他调出正式通知，一行一行看下来。路径一，路径二，路径三。他没有点开那条陌生消息里的附件，也没有给苏棠发消息问她的信息来源。他选了路径二——那条看起来最常规、所有人都知道的陷阱最多的路。

键盘声响起来的时候，他心里有另一个声音。

选这条路不需要跟任何人解释。不需要承认他收到过一条不该收到的消息，不需要承认苏棠的信息可能比他的更有用，不需要承认他刚才在茶水间里沉默的那几秒钟是在害怕——害怕开口意味着再次把信任交到别人手里，意味着他欠她的东西又会多一层。

他怕的不是欠。

他怕的是有一天她发现，他之所以一直欠着，是因为他不敢还。

方案第一稿写到一半的时候，窗外的天开始暗了。四月的天黑得很快，像是有人在天上拉了一道帘子。办公区的灯一排一排亮起来，荧光灯管发出细微的电流声。林延活动了一下僵硬的肩膀，继续敲键盘。

他不知道此刻苏棠正坐在楼下食堂的角落里。面前搁着一碗已经坨掉的面条，筷子斜插在碗里，像是被人拿起又放下过很多次。她手机屏幕上亮着一条对话框，赵锐的头像是灰色的——不在线，或者故意不让她看见在线状态。

消息内容只有一行：「林延选的是路径二。」

苏棠盯着那行字看了很久。拇指悬在键盘上方，打了一行字又删掉，删掉又打。

最后她发了四个字：「你怎么知道。」

赵锐没回。

食堂里的电视在播晚间新闻，声音开得很低。苏棠把面条碗推到一边，拿起保温杯喝了一口。枸杞水已经完全凉了，凉得舌尖发涩。她看着手机屏幕暗下去，没有再解锁。

她想问他为什么不来找她。想问他是不是不信任她拿到的那条消息。想问他在茶水间里沉默的那几秒钟，脑子里到底在想什么。

但她也没问。

有些事情一旦学会了不问，就很难再学会开口。就像某些信任一旦被借过一次，就很难再被当作理所当然的东西放在两个人中间。

走廊里，公告栏上的那张A4纸还在风里一掀一掀的。路径二的截止日期印在右下角，黑色的宋体字，干净利落，像是某种早已写好的答案。
