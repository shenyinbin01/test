# 第三章 代价

第三关的通知没有发邮件。沈主任站在会议室门口，挨个叫人。

林延是第三个被叫到的。

会议室里的空调开得很足，比走廊低了不止五度。沈主任坐在长桌对面，面前摊着两摞材料——左边是林延前两关的方案和评分，右边是他不认识的一份东西，封面上贴着机密封条。

“坐。”沈主任没抬头。

林延坐下。椅子是折叠椅，坐上去的时候关节发出一声短促的金属响。

“第一关你选了路径二。第二关你的协作对象替你完成了百分之八十的方案框架。对不对。”

“对。”

沈主任从左边那摞材料里抽出一张纸，翻过来放在桌上。是一份评议记录，上面有几行手写批注。林延认出了其中一行——路径选择保守，协作主动性不足。

不是第一关的评语。是综合评语。

“三关考核的最终评审，看的不是某一关的分数，”沈主任摘下眼镜擦了擦，“看的是三关加在一起，你表现出了一种什么样的行为模式。”

林延的手放在膝盖上，掌心有点湿。

“你的模式是，”沈主任把眼镜戴回去，“在可以选择对外依赖的时候，你选择对内消化。在可以共同承担的时候，你选择独自承担。表面上看这是不麻烦别人。从组织角度看，这是协作信任的缺失。”

林延想反驳，但他想起苏棠在走廊里说过的那句话——你信我但你不敢说，那跟不信有什么区别。

他没有反驳。

“但第三关还没有走完，”沈主任把右边那份密封材料推过来，“这一关叫最终评估。不是方案评估，是你需要当着评审组的面做一个决定。”

材料拆开，里面只有一张纸。

上面写着一个问题：你愿意为晋升放弃什么。

不是比喻。不是心理测试。下面是三行选项，每一项后面都标注了评分权重。第一项：放弃本次晋升资格，保留下一轮优先权。权重中等。第二项：接受降半级晋升，工资等级下调一档，但立即生效。权重偏低。第三项：保留全部晋升资格，但以书面形式公示你的不足项，由你的协作对象签字确认。

林延看到第三项的时候，手指在纸面上停住了。

协作对象签字确认。苏棠。

沈主任观察着他的表情，补了一句：“不是要她替你担保。是要求她确认你的不足之处真实存在。确认之后，这些不足会写进你的晋升档案，保留三年。不影响晋升，但影响下一次评估的起点。”

林延沉默了。

他想起第二关结束那天晚上，苏棠发来的那个框架。她一个人把方案搭完了。她给的“不需要”不只是在拒绝他，也是在用她自己的方式替他分担——多做就少犯错的余地，她把那份余地塞给了他。

现在评审组反过来把这道题摆在他面前：要不要让她签字。

如果他选第三项，他晋升成功了，但档案里会永久记录他的协作信任不足。苏棠的签名会成为那份记录的一部分。她签下去的时候在想什么——他不知道。

如果她不签，他也可以选第二项，降级晋升。工资少一档，但档案干净。或者选第一项，放弃这次，等下一轮。

哪一个都不干净。

“你有十五分钟。”沈主任说。

林延低下头，看着纸上的三个选项。会议室里安静极了，只有空调出风口的嗡嗡声和沈主任翻动纸张的轻响。

他想起三个月前苏棠的导师在部门会上说的那句话。想起公告栏前那条陌生消息。想起茶水间里苏棠搁下保温杯的声音。想起第二关走廊里她的肩膀擦过他手臂的触感。

然后他想起今天早上。他在办公楼下碰到了苏棠。她在接水，保温杯拧开盖子搁在饮水机边上。他走过去的时候她抬起头看了他一眼，什么话都没说，端着杯子走了。但她走之前，手指在饮水机的按键上多停了一秒——替他按下了热水键。

水还在流。她已经走了。

林延把笔拿起来。

他在第三项旁边打了一个勾，然后把材料翻过来，在空白处写了一行字。字迹不整齐，像是手在用力控制着不要抖。

「我愿意接受公示。协作对象签字前，我会亲自跟她说明原因。」

沈主任接过材料看了一遍，没有评价。他只说了一句：“评审组会在今天下班前给你答复。”

林延走出会议室的时候，走廊里的感应灯亮了一排。他的后背全是汗，衬衫黏在皮肤上，空调的风一吹凉得刺骨。

他走到苏棠的工位。她不在。电脑屏幕黑着，桌上的保温杯还冒着热气，说明她刚走不久。

他从笔筒里抽了一张便签纸，写了几行字，压在保温杯下面。

「评审组让我选晋升代价。我选了需要你签字确认的那一项。不是因为我想让你替我担什么。是因为我想让你知道这次我选了开口，虽然开口的方式可能不对。你可以不签。我已经做好降级晋升的准备。但我想让你知道。」

他把便签纸压好，转身走回自己的工位。

三点半。四点半。五点半。

下班铃响的时候，沈主任的邮件来了。

标题：第三关评审结果。

林延点开邮件，手指在触摸板上顿了一下。评审结果只有三行：

「林延，晋升通过。综合评级：合格。档案记录：协作信任不足，由协作对象苏棠确认。」

下面是苏棠的电子签名。

她签了。

邮件末尾有一行附注，是评审组统一加的：「晋升结果即日生效。档案记录保留三年，作为下一轮评估参考。下一年度协作能力须有实质改善。」

他靠进椅背，盯着那行附注里的最后一个分句。下一年度协作能力须有实质改善。评审组在告诉他：你这次过关了，但代价是下一次的门槛更高了。

鼠标往下滑。邮件底部还有一行小字，不是评审组的格式，是苏棠在签名备注里加的：

「签了。不是因为你写的那张便签。是因为第一关你选了路径二的那天下午，我在食堂里坐了两个小时，一直在等你给我发一条消息。你没发。但今天你发了。三年档案换一条消息，不算贵。」

林延把这段备注读了三遍。

不算贵。

他笑了一下。不是开心的笑，而是一种堵在喉咙里的东西终于松开了一点的笑。但因为备注里的每一个字都在告诉他同一件事：他欠她的，又多了。

上次欠的是信任的债。这次欠的是签名的债。下次欠的是什么，他不知道。但他知道只要这种模式不改变，他就会一直欠下去。

而苏棠已经不是在等他还不还了。她是在看他下一次开口的时候，是不是还要她自己先走过来。

走廊里的灯又灭了一排。加班的人陆陆续续走了，整层楼安静下来。林延把邮件打印了一份，折叠整齐，放进抽屉最里面。然后他拿起手机，打开跟苏棠的对话框。

上一次对话还停在她那句“不需要”上。

他打了几个字，停下。删掉。又打。最后发出去的是很简单的一行：

「谢谢你。下次我不会让你先走过来。」

苏棠的回复来得很快。只有一个字。

「好。」

林延把手机放下，屏幕暗掉之后他看见自己的脸映在黑色玻璃上。表情很平，但眼睛里有一点光，不是如释重负，是一种终于被看见了一部分的清醒。

晋升通过了，档案里多了一条三年的记录，苏棠的签名安静地躺在数据流的最底层。下一次评估的门槛抬高了一截，下一段关系里他欠她的又多了一层。

这不是干净的晋升。但也许有时候，不干净地往前走，比干净地站在原地更接近答案。
