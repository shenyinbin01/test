# 第三章

第三关开始之前，林延在走廊里找到了苏棠。

她没有进等候区。她站在走廊尽头的窗边，背对着他，在看外面已经黑透的天。窗外是院子里那排老樟树的树冠，夜色里分不清叶子和枝干的边界，只能看见一团深色的轮廓在风里微微晃动。走廊里没有别人，头顶的日光灯发出平稳的低频嗡鸣。

林延走到她身后两步远的位置，停了下来。

"第一关。"他的声音在走廊里显得比平时单薄。"进门前，我在走廊里听到了一些话。"

苏棠没有转身。

"有人说沈主任要测的不是对错，是信什么——信正式渠道，还是信自己的判断。"他停顿了一下，手指在腿侧绷紧然后慢慢松开。"我应该当时就告诉你。应该在你问我'沈主任的意思是什么'的时候，就把这句话给你。"

这句话他说出来了。语气比他预想的平。太平了。像是在陈述一份事后补充的报告，不像在弥补一个人对另一个人沉默了两天的事。他自己也听见了。他可以控制字词，但他控制不住字词之间那种缺少温度的空隙。

苏棠转过头，看了他一眼。中间隔了大约两次呼吸的时间，然后她转过来——身体没有动，只是头颈转了一个角度，让视线从窗外的树冠移到他脸上。

然后她点了一下头。

很轻。下巴往下沉了不到一寸，然后收回。没有追问"谁说的"，没有问他为什么沉默了两天，没有问他为什么现在才开口。那个点头意味着她接收了信息——事实层面的信息——但到此为止。

"知道了。"她说。

她从他身边走过，肩膀和他之间隔了和任何一个同事没有差别的距离。她推开等候区的门，走了进去。门在身后合上。

林延站在原地。他忽然意识到一件事。他给苏棠的是一个缩水版本的真相——他把"说了什么"告诉了她，但他没有说那条传闻对他的意义。他没有说他沉默是因为害怕在她面前变成需要借别人的话来补自己判断力的人。他没有说。而苏棠的"知道了"告诉他，她知道他没有说。

但他已经没有时间重新开口了。等候区的门完全打开，沈主任站在门口。

"第三关——最终评估。所有人进来。"

大厅里没有隔板。三张桌子呈品字形排列，沈主任的座位在上方平台上。每张桌子上放着一份文件，封面印着"综合评审"四个字，和第一关通知里的"信息辨识"出自同一个人的措辞习惯：简洁，不含解释。

沈主任没有坐下。他站在三张桌子前方，目光从三个人脸上依次掠过，节奏和第一天发通知时一模一样。

"最终评估的内容很简单。你们每个人需要陈述自己在前面两关中的判断依据——基于什么信息，做出了什么选择。可以开始了。"

赵锐第一个起身。他的陈述利落干净——所有判断基于正式通知文本，不涉及正式渠道以外的信息。他说完后坐下，双手交叠放在桌面，像是提交了一份已签字的文件。

苏棠第二个。她的陈述同样简洁，但用词不同。她没有说"正式渠道"，她说了"独立判断"。她说到"独立"两个字时，音调没有变化，但在那个字和下一个字之间留出了一个几乎察觉不到的空隙——短到不能称为停顿，但长到让这个字在空气里多悬了一瞬。

轮到林延。

他站起来，低头看了一眼桌面上的文件，然后抬起头。

"我在第一关开始前，从非正式渠道收到了一条信息。这条信息影响了我对前两关的判断。"他的声音比他自己预期的要稳。比在走廊里对苏棠说话时更稳。人在面对机构时有时候比面对一个人时更容易说出真相。"我没有在第一时间与搭档共享这条信息。这是我的判断失误。"

沈主任的目光定在他身上。"什么信息？"

"关于第一关测试重点的传闻。传闻称第一关测试的是信息辨识能力——辨识自己应该信任哪种判断依据——而非简单的对错判断。"

沈主任看了他三秒。然后拿起笔，在面前的记录本上写了几个字。笔尖划过纸面的声音和第一天推送文件时一样清晰。

"记录在案。你可以继续。"

林延继续陈述了第二关的逻辑——依据有限信息选择了进取方向。他说到"有限信息"时没有解释为什么信息有限，没有提苏棠的隔板和赵锐的试探。他的陈述在事实层面完整，在原因层面是空的。他知道。沈主任知道。苏棠也知道。

三十分钟后，沈主任合上了记录本。"结果在下班前公布。"

林延独自在座位上坐了一会儿。桌面上有一道前一个使用者留下的浅痕，他伸手摸了摸，指尖下木头的凉意和第一天接过通知时纸边的温度一样。

他通过了。他不知道自己是怎么知道的，但他知道——那个"记录在案"不是否决，是备注。他可以通过三关但仍然被标记为一个曾经依赖非正式渠道的人。这就是代价。他能承受的那部分代价。

真正的代价在走廊里。在苏棠的点头里，在她陈述的"独立"二字里，在她从他身边走过时不变的距离里。他把事实给了她，但事实不是她等了整整两天要的东西。她要的是他为什么沉默。他没有给。他不知道该用什么语气去说那个理由。

下班前，结果公布了。

公告贴在公共区的布告栏上，和第一天通知用同一种纸张、同一种宋体。三行字：林延——通过，备注。苏棠——通过，无备注。赵锐——通过，排名第一。

林延在布告栏前站了一会儿，然后把双手插进口袋。右手指腹碰到了那份通知的折痕。折了三天的纸，边缘已经完全毛了。

走廊尽头，苏棠正从另一个方向走出来。她抬头看见了他。

两个人隔着半个走廊的距离，同时停了一步。不是猛然的停——是步速自然地降到零，像两个人在同一瞬间想起一件还没处理完的事。

苏棠微微侧了一下头。幅度很小——和她之前的点头差不多。她的嘴唇动了动，像要说什么。但她没有说。她收回视线，继续朝她的方向走了。

林延看着她的背影在走廊尽头拐弯，消失。

他把手从口袋里抽出来。手指还保持着捏着通知折痕的姿势——拇指和食指弯曲，中间隔着纸的厚度，但纸在口袋里，手指之间什么也没有。他把通知掏出来，展开。折痕在纸面上形成了几条交叉线，把"信息辨识"四个字恰好框在中间的一个小格子里。

他看了一会儿，然后把纸重新折好——沿着原来的折痕，不是新的——放回口袋。

然后他朝自己的方向走了。走廊很长，脚步声在日光灯的嗡鸣里有节奏地响着。

没有人回头。
