# 第一章

通知是下午三点到的。

沈主任站在长桌尽头，将三份文件依次推过桌面。纸页划过木纹的声音很轻，但房间里没有别的声音——没有人翻页，没有人喝水，连椅子的转动都停了——所以那声音显得格外清楚，像有人用指甲在桌面上划了一道浅痕。

"三关评估。"

沈主任的语速不快。每个字之间都留着等距的停顿，让人分不清他是在给时间消化，还是在预留一种看不见的测量空间。

"每一关测试的内容不同。第一关——信息辨识。"他的目光从三个人脸上依次掠过，不偏不倚，在林延那里停的时间不比苏棠长，也不比赵锐短。"你们的判断力会在那里接受检验。判断的依据，和判断的结果，同样重要。"

林延伸手接过自己的那份文件。纸是普通的A4复印纸，但指尖触到边缘那一瞬的凉意让他下意识收了一下手指。他没有立刻低头去看内容。他先把文件平放在面前，用手掌压了一下封面，感觉到纸页下面桌面的硬度。

沈主任说完便离开了。皮鞋声沿着走廊由近及远，响了几下，然后彻底消失。房间里重新安静了几秒，接着赵锐最先起身，将文件夹在腋下走了出去。然后是苏棠。她站起来时椅子后退的声音比赵锐轻一些。她看了林延一眼——不是注视，是那种经过一个人时自然会产生的视线接触——然后也出去了。

林延是最后一个离开的。他拿着文件走出来时，走廊里已经空无一人。

走廊的另一端，有人叫住了他。

是周岩。隔壁组的，平时不怎么打交道，属于那种在食堂碰见会点头但不会坐下来一起吃的关系。周岩从侧面的一扇门里探出半个身子，左右看了看走廊两端，然后朝林延侧身靠近。他的声音压得很低，低到像在用气息说话而不是喉咙。

"林延。第一关——别只看通知上的东西。"

林延没有说话。他站在原地，感觉到周岩的呼吸在他耳边制造了一小片热区。

周岩的目光越过林延的肩膀，扫了一眼走廊尽头——那是沈主任刚才离开的方向。然后他补充了后半句，声音更低了一层："听说沈主任要测的不是你对错，是你信什么——信正式渠道，还是信你自己的判断。"

他拍了拍林延的肩，力道不重，但落手的位置恰好是锁骨上方那一块，让人无法忽略。然后他缩回门里，门在他身后合上了，发出轻微的咔哒声，像是锁扣归位。

走廊重新安静下来。日光灯在头顶无声地亮着，把墙壁映成一片均匀的白色。

林延站在那里，手里的文件忽然变得比刚才重了一些。不是真的重了——纸还是纸——是他拿着它的方式变了。他不再用三根手指夹着边角，而是用整只手握住文件的中间位置，像是怕什么东西从纸页之间滑出去。

他把文件折好，放进口袋。折痕在口袋里硌着他的大腿外侧。

回到公共区时，苏棠已经在等他了。

她坐在靠窗的位置。面前摊着同样的通知文件，用红笔在某些段落上做了标记——不是圈注，是那种从段落开头划到结尾的细直线，整齐、克制。窗外的光线落在纸面上，让红笔的痕迹看起来比实际上更醒目。

看见林延进来，她抬起头。动作不大，只是下巴往上抬了一点，视线从纸面上移到他脸上。

"你怎么看？"

她的声音和平时一样——平稳，不紧，但也不松。问的不是"你看了吗"或者"你觉得难不难"，而是"你怎么看"。一个需要回答的问题，不是一个可以敷衍的招呼。

林延拉开椅子坐下。椅子的金属腿在地面上刮出一声短促的摩擦音。他把目光落在自己那份通知上，翻开封页，扫了几行字，没有去看苏棠标记过的段落。

"按流程走就行。"

他听见自己说出这句话。声音是正常的。他不确定它听起来有没有他感觉到的那么空。

苏棠的笔停了。

那种笔尖离开纸面的停顿——不是写完了停下来，是写到一半，笔还在手里，但不再往下走。持续了一秒，然后她继续在纸上划了一道线。这条线比之前那几条稍微用力了一些，红笔的墨迹在结尾处洇开了一个很小的点。

"第一关考信息辨识——这个措辞不太像常规的笔试。"她把笔搁在纸边，抬起头看着他，语气平稳，像是在陈述一个事实而非提问，"你觉得沈主任的意思是什么？"

林延的手指在桌面上无意识地滑动了一下。指腹擦过木质表面，触感干燥，微微粗糙。他在想周岩的话。周岩的声音还在耳朵里——别只看通知上的东西，测的是你信什么——但他没有办法把那两句话说出口。

说出来意味着什么？意味着他站在走廊里听了一个几乎不认识的人告诉他该怎么理解评估，然后在进入正式测试之前就已经开始掂量要不要依赖这条信息。他不想让自己变成那样的人。一个需要借助走廊耳语才能往前走的人。至少不能在苏棠面前变成那样。

他和苏棠之间一直有一种没被明说过的平衡。她帮过他。不止一次。他欠着她的信任——那种信任不是你开口要的，是她主动放在那里的，像共享一把备用钥匙。他不能让她看到他在借用别人的暗示，因为那等于告诉她：你给我的东西不够用。

"通知写什么，我们就准备什么。"他说。

苏棠的笔又停了。

这一次停得更久。她没有继续划那条线——她的手搁在纸面上，笔夹在食指和中指之间，笔尖悬在离纸面半寸的位置。她转过头看着他，目光平和，但比刚才多了一层东西：不是质问，不是失望，是一种安静的测量。她在看他，像是在看他会不会自己补充点什么。

他什么也没补充。

她合上文件。动作很轻，纸页合拢时几乎没有声音。

"那我先回去看了。"

林延点了点头。

她没有立刻转身。她站在那里，手搭在椅背上，食指在椅背的塑料包边上轻轻敲了两下。嘴唇动了动，像是还有一个问题要问——一个在她开口之前就已经被他挡住的问题。但她没有问出来。她的手指从椅背上滑下来，垂在身侧握了一下，松开。

她走了。

林延听见她的脚步声在走廊里响了几下——节奏比平时慢了一点，每一步之间多隔了不到一秒，但就是那不到一秒让步伐听起来不像走，像在等什么。然后脚步声消失了。

公共区里只剩下他一个人。

他把手伸进口袋，触到了那份通知的折痕。纸已经在他的体温下微微变暖。周岩的话还在脑子里转，像一段被卡住的音频，反复放同一句：测的是你信什么。他应该告诉苏棠的。他应该在她合上文件之前把那两句话递过去，像她曾经把备用钥匙递给他一样。但他没有。他不知道说出这句话之后该怎么面对她接下来的问题——"谁说的？""你相信吗？""你为什么不早说？"——每一个问题都在问他同一样东西：你是不是觉得自己不够。

窗外的光线正在变暗。桌面上的红笔痕迹渐渐失去了白天的锐度，变成几条模糊的细线。

他翻开通知，从头到尾读了一遍。沈主任的措辞确实奇怪。"信息辨识"四个字被放在段落的中间位置，前面没有修饰语，后面没有解释。正常的文件不会这样写。正常的文件会说"请根据以下材料判断选项的正误"或者"测试将对信息分析能力进行评估"。沈主任没有。他只给了四个字。

他把通知重新折好，放回口袋。折痕更深了一折。

明天就是第一关。他不知道苏棠会怎么准备。他只知道他没有告诉她的东西已经变成了他们之间的一个间隙——现在还很窄，窄到可以假装它不存在，窄到在他和她说话时可以用"按流程走"绕过去。

但间隙这种东西，一旦有了，就不会自己消失。它只会被下一次对话，或者下下次沉默，撑得更宽一些。

走廊尽头的日光灯闪了一下，然后重新稳定下来。

林延站起身，朝门口走去。走出公共区时，他下意识回头看了一眼苏棠坐过的那张桌子。椅子上已经没有她了。桌面上留下了一支红笔，笔盖没套回去，笔尖还露在外面。
