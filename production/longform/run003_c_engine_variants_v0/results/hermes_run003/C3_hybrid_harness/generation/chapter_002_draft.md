# 第二章

第二关的门和第一关不一样。

第一关是一间普通的会议室——长桌、文件、沈主任端坐在尽头的注视。第二关是一间被灰色隔板分割成数个独立区域的大厅。每个区域大约两米见方，里面摆着一张窄桌、一把没有扶手的椅子和三个不同颜色的文件夹。墙壁上贴着用编号标记的流程说明，字体和第一天通知上的完全一致。没有沈主任坐在面前，但头顶对角线上各有一台摄像机，红灯亮着，稳定地、不眨眼地亮着。

林延站在入口处。手里的任务单薄薄的，只有一页。

任务单上的正文占了两行。第一行：根据已有信息确定下一步行动方案。第二行下面是三个选项，每个选项后面跟着一个字母和一个数字——A7、B3、C5。没有解释A、B、C分别代表什么，没有说明数字是权重还是编号还是别的什么。沈主任在第一关测试"信息辨识"，在第二关留下了一个需要辨识的信息缺口。

林延看了一眼选项，又看了一眼。

周岩在走廊里说的那句话——测的是你信什么——此刻在他脑子里变成了一个无法使用的工具。像一把钥匙，你知道它能开某扇门，但你不知道是哪一扇。

三个选项分别指向三种策略方向：A7保守，按正式通知的内容走，不求加分但求不减分。B3进取，在通知信息之外自行扩展推断，回报可能更高，但推断错了代价也更大。C5取中，选最不容易出错的那条路，但同时也是最容易被淹没的路——你不犯错，但你也什么都没证明。

如果周岩说得对——沈主任真正要测的不是对错而是判断依据——那么A7可能会因为"缺乏独立判断"而被标记。B3的推断方向如果不在沈主任的评分框架里，就是自己往错的方向跑了。C5似乎最安全，但在一个测试"判断力"的关卡里，什么也不判断可能本身就是一种失败。

他需要知道规则。但他手里的规则只有周岩的半句话。半句话。

林延低下头，在三个选项之间反复切换视线。纸面上的字母和数字没有任何变化。他把任务单翻过来——背面是空白的。沈主任没有给更多的说明。

身后有脚步声。不止一个人的。

苏棠从另一个入口进来，已经领了任务单。她的步速比平时快了一点，不是慌张，是已经有了方向的那种快。她没有朝他这边看，径直走向属于自己的隔间，坐下，翻开文件夹，拿起笔。从林延的角度只能看到她的侧影——肩膀的线条平直，手臂放在桌上时没有任何多余的移动。

林延看着她的背影，想叫住她。

他想说的话在喉咙口卡了两秒。比昨天那句更难。昨天他可以说"按流程走就行"——那是一句什么也没给的句子。今天他需要从她那里拿到东西——她的判断，她读文件的方式，她对那三个字母数字组合的理解——但他没有立场开口。昨天他关上了门。今天他站在门外。

他最终没有叫她。

隔板把他和苏棠分成两个独立的方格。他能听见她那边纸张翻动的声音，听见笔尖落在纸面上的沙沙声，频率均匀，节奏稳定——不像一个在选项间犹豫的人。她已经做了决定。不需要他，也可以做决定。

林延收回视线。他选了B3。进取选项。没有足够信息支撑的进取，像闭着眼睛伸手去摸一堵看不见的墙，不知道墙那边是地板还是深渊。

二十分钟过去了。隔板另一侧，苏棠的纸声停了，然后是文件夹合上的声音——厚纸板碰在一起发出的闷响，干脆利落。椅子被推开，脚步声从他的隔间前方经过，没有停顿。

林延低头看了一眼自己的进度——还剩将近一半。他的手在纸面上压了一下。

赵锐从侧门走了进来。

他的入场方式和苏棠不一样。不快，不慢，不快不慢本身是一种信息。他手里拿着任务单，垂在身侧，路过林延的隔间时，步速没有变化，但步幅——在经过隔板开口那一瞬——收短了一点，恰好够他的肩膀在开口处停留一秒。不是停下来看。是恰好经过时，他和林延之间只隔了一块隔板的厚度。

"走廊里的话，跟门里的东西，不一定对得上。"

赵锐的声音不高。不是悄悄话，但经过刻意控制，只有隔板这边能听见。像随手丢下一枚硬币，听它在地上转不转。

然后他走了。步速恢复，步幅恢复。他进了自己的隔间，和所有人之间都保持着一模一样的距离。

林延握着笔的手指在纸面上按了一下。笔尖压出一个墨点，很小，但纸洇了，墨迹在纤维之间扩散成绿豆大的一小块黑。他不知道赵锐是什么意思。

是听到了周岩的耳语？周岩说话时走廊是空的，但"空"这件事在林延的确认里有多可靠？是赵锐自己也听到了同样的传闻，还是听到了别的什么？或者赵锐根本没有信息——没有周岩，没有传闻，没有任何走廊里的耳语——他只是在试探。试探林延的反应。试探林延在第一关结束后的沉默到底藏着什么。一个人如果守着一个秘密，他面对一句不指名的话时，那个秘密会替他做出反应。赵锐不需要知道秘密是什么，只需要知道秘密存在。

林延无法确认赵锐是哪一种。这才是最糟糕的部分。你无法确认。

隔板另一侧，苏棠站了起来。

林延听见椅子被推开的声音——金属腿在地面上刮过，短而干脆的一声。然后是脚步声，朝他这边来的，在隔板外侧停了一下。透过隔板和地面之间那条不到两厘米的缝隙，他看见她的一只鞋尖。黑色的平底鞋，鞋尖朝着他的方向。

"你自己可以的。"

她的声音和昨天一样平稳。音量不大不小，语气不冷不热，像是在陈述一个经过核实的客观事实。和昨天他在公共区对她说的那句话——按流程走就行——用的是一模一样的结构：一个笼统的判断，不给细节，不留入口。

然后她的鞋尖转了方向。脚步声渐远，出了大厅。

林延盯着面前的选项纸。选项还是那三个选项，字母和数字一个也没有变。但他忽然意识到了一件事。苏棠昨天想从他这里拿到的东西——他对沈主任措辞的真正判断——他没有给。今天他想从苏棠那里拿到的东西——她如何理解三个选项的暗码——她也没有给。他们之间形成了一个对等的沉默，像两面镜子互相照着，正面没有东西，背面互相知道。

从大厅出来时，天色已经完全暗了。

走廊里只亮着应急灯，在墙角低处映出窄窄的一条白光。林延沿着走廊往外走，步子比进来时慢。他口袋里的通知文件还在——第一关的，折了三次，边缘已经开始泛出灰白色的毛边。他走出大楼时，夜风迎面过来，带着外面院子里不知哪棵树的气味，凉的，有一点湿。他站了一会儿，感觉风把手里的汗吹干了。

明天是最后一关。他需要在那之前做一件事。一件他已经晚了整整两天的事。但他还不确定自己能不能做到——他不确定的是，做到之后，苏棠会给他什么回应，或者什么都不给。

他只知道选项纸上的B3已经交了。他没有退路。隔板另一侧的那句"你自己可以的"，会在今天夜里反复响起，不带任何多余音调地、平稳地响起。
