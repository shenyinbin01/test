# 第一章 不完整的消息

会议室的白炽灯管嗡嗡响了两声，像某种不确定的预告。

林延把刚收到的正式通知在桌面上摊平。纸张边缘微微卷起，红色的公章盖在落款处，字迹清晰——三门考核，第一门明日启动。通道编号、签到时间、注意事项，每一项都印得一丝不苟。他盯着那行"信息筛选——三分钟决策窗口"看了很久。

"收到没有？"苏棠的声音从背后传来，带着一贯的轻快。她刚从走廊那头跑过来，额角还挂着训练场的薄汗。她把椅子拖到林延旁边坐下，目光落在那张通知上，嘴角动了动，"跟我的一样。沈主任这次发得挺准时。"

林延点了点头，把通知折好收进口袋。桌面空出来一块，茶水杯底在塑料贴面上印出一圈水痕。

苏棠没注意到他的动作。她已经开始盘算第一门的路线，手指在桌面虚画着些什么。"咱们得走正式通道的东西两侧，信息筛选那关我查过前三年的案例，靠前的候选人大多——"

"苏棠。"林延打断她，声音比预想的轻。

她停下来，看着他。那种直视人的习惯是她跟了他两年养成的——每一次确认信息，每一次核对判断，都是这种眼神。不闪不避，等他说完。

林延想说的话在喉咙口转了一圈。他想告诉她下午他在茶水间门口听到的东西：一个压低的声音提到第一门考场的西侧廊桥有维修梯可以绕开签到记录，说这是"内部安排"，信不信由你。说这话的人是赵锐，竞争对手的身份摆在明面上——赵锐站在茶水柜旁边，背对着门口，声音压得刚好能让经过的人听见。林延没进去，站在门框后面听完了后面两句。"不走正式通道的人会吃亏，"赵锐说，"但走正式通道的人可能会错过时间窗口。"

这话真假参半。让人不安的正是这个"参半"。

但他说不出口。

不是不信任苏棠。是另一种更私密的犹豫——说出来，等于承认他在意这种旁门左道的消息，等于承认他觉得正式通道不够。苏棠会怎么想？她会觉得他想走捷径，还是觉得他不相信她的判断？她花了那么多时间整理案例，每一个失误点都标注得清清楚楚。她相信正式通道。她从来没问过"有没有别的路"。

"林延？"苏棠的眉毛微微扬起。

"没事，"他说，"路线的事明天再说，今天先把基础材料看一遍。"

苏棠看了他两秒，没有追问。她收回手，从背包里抽出一叠打印稿推过来。"也行，你爱先看材料。这是我整理的全部信息点，包括三年前那几个案例里的失误点。"

她做了全部功课。每一个案例，每一个失误点，都标注得一清二楚。她对他没有任何保留。

林延接过那叠纸，指尖在纸面上划过去。印刷体的笔画清晰而沉默。他口袋里的通知压在折痕上，像一块小石头。

"苏棠。"他开口，声音有些发紧。

"嗯？"

"没什么。"他把打印稿翻开，"好，我开始看。"

苏棠偏过头看着他，嘴角的弧度维持了一会儿，然后慢慢收平。她没再说什么，站起身走到窗边，背对着他，手指在窗台上轻敲了两下。

那两下敲得极轻。林延却觉得自己的肋骨被什么东西撞了一下。

他低下头，假装在看第一页的材料。字在他眼前浮着，他的注意力却全部集中在口袋那一小片硬纸卡的重量上。

赵锐那句话还在他脑子里转："信不信由你。"

信，还是不信？这件事本该是他和苏棠一起判断的。现在他把它拆成了两半，一半藏在口袋深处，另一半留在桌面上，假装从未见过。

窗外有风穿过走廊，把门帘掀开一角。苏棠的侧脸在日光灯的冷白光里显得比平时安静。她没有回头。

林延翻到了第二页。这页上标注的是往年第一门淘汰率最高的环节——信息初筛，三分钟判断窗口。苏棠在旁边用铅笔写了一条批注："对方给出的信息完整度会低于你预期，不要假设对方会补全。"

不要假设对方会补全。

他重新折好口袋里的通知，折痕又深了一层。

苏棠终于转过身来，走回桌边拿起自己的杯子。她喝水的时候看了他一眼，那眼神和平常不太一样——不是质问，而是一种等待。她等过他很多次，等他从分析里抬头，等他从犹豫里走出来。每一次她都等了。

这一次她没等到。她放下杯子，背上背包。

"明天七点，东门见。"

"好。"

她走到门口，脚步顿了一下，像还有什么话没说。林延看见她的肩膀微微收紧，然后松开。门已经推开了，走廊里的风涌进来，吹得桌上的纸张沙沙翻动。门在她身后合上的声音沉稳而干脆。

林延一个人坐在会议室里，把那叠打印稿从头翻到尾。每一个标注他都看进去了，每一处苏棠画下的下划线他都能感受到指力。第三页的末尾有一条他没注意到的铅笔字，写得很轻："如果你有别的判断，告诉我。"

她写的时候大概犹豫过力道。

但他下午听到的那句话，他始终没有写下来。没有标注，没有标记，没有在任何地方留下痕迹。

他把这一切归因为"需要自己先确认"。但确认给谁看呢？他不敢回答这个问题。

窗外的天色沉了下去，日光灯管的嗡嗡声重新回到他的听觉里。明天七点，第一道门会准时打开。

他口袋里有两样东西：正式的入场通知，和一条他没跟任何人提起过的消息。

哪一样会先被拆开，他心里没有答案。

走廊尽头的挂钟敲了整点。夜色从窗外漫进来，把会议桌上的水痕浸成深色。
