# 第二章 错位的回音

第一门通过的消息来得比预想的平静。

沈主任的助理在走廊公告栏贴出名单时，林延排在第四。苏棠排第二。赵锐排第一。名单上没有分数，没有评语，只有顺序。林延看了三遍才确认自己的位置——不高不低，刚好够进入第二门。

苏棠站在他旁边，盯着赵锐的名字看了很久，没说话。

他们走向第二门考场的路上，苏棠终于开口："第一门的信息筛选，你用了什么判断？"

她问得很直接，像往常一样。但语气里多了一点东西——从他昨天下午在会议室说"没事"那一刻起，她就一直在等。

林延知道这一次不能不回答。沉默在昨天是第一门的选择，今天进第二门，规则已经变了——重复使用同一种应对方式会被系统自动标记为"策略固化"，直接扣分。他看过通知背面的细则：每一门考核都会记录候选人的决策模式，重复模式视为缺乏应变能力。

"我用了正式通道的全部信息点，"他说，尽量让声音平稳，"没有偏航。"

这句话本身是事实。他没有使用赵锐提到的维修梯。他走了正式通道。但这句话不完整——他藏了一个"为什么"。

苏棠的脚步慢了一拍。"全部？"

"全部。"

她看了他一眼，那一眼持续的时间比平时长。林延知道她在算——他们两个用了同样的正式通道，他排第四她排第二，这中间的两名差距来自哪里？

来自判断窗口里的某个微调，来自筛选环节的一处取舍。来自他脑子里多出来的那条消息——即使他没使用，它也占了他的判断空间。

第二门的考场设在一间独立隔间里，四面白墙，中间一张长桌，桌上放着一个没有标记的通讯终端。终端屏幕上滚动着一条实时指令："请与同组人协调路线。每人输入一次路径选择，不可修改。路线冲突则双方扣分。"

苏棠已经坐到终端另一侧，手指悬在输入区上方。

"林延，我们直接同步。"她说，声音干净利落，"我选B线，你也选B线。看过案例了，B线是最优解。"

林延的目光在屏幕上扫了一遍。屏幕上除了输入区，还有一条灰色的辅助信息条，上面写着："系统提示：A线存在未公开的优先通道，需辅助身份验证。"

辅助身份验证。

这条信息没有发给苏棠。她的终端上不显示这条提示——林延从她的表情判断，她只看到了标准界面。系统在给他们不等量的信息。

这是第二门的测试。不是测试谁能找到最优解，而是测试在信息不对等的情况下如何协调。赵锐坐在隔壁隔间，他的屏幕上可能也有不同的提示。整个第二门的结构像一面碎镜子，每个人只拿到一块碎片。

"苏棠，"林延的声音压得很低，"你屏幕上有没有看到——"

"什么？"

他停住了。昨天的场景像倒带一样涌回来——会议室，日光灯，苏棠在窗台上敲的那两下手指。

昨天他选择不说。今天如果他说了，等于承认昨天的沉默是有代价的。等于承认他欠她一个解释。

"B线，"他说，"我也选B线。"

他输入了B线。手指按下确认的时候，指腹在按键上多停了一秒。

系统处理了三秒钟。然后屏幕上弹出一行红字："B线与另一组人冲突，双方均扣三分。"

林延愣了一下。他抬头看苏棠，她的脸上没有意外。她甚至轻轻呼出一口气，像一件早已预料的事情终于发生了。

"你知道会冲突？"他问。

苏棠把终端屏幕转过来给他看。她的界面底部有一条他没见过的提示——绿色的辅助信息条，写着："协调建议：对方终端显示A线优先通道提示。如对方放弃A线选择B线，建议本方配合选择B线以测试对方信息共享意愿。"

她从一开始就知道。

林延看着那行绿字，感觉自己的判断像积木一样一块一块掉下来。苏棠拿到了关于他的提示，知道他的屏幕上有一条灰色信息。她一直在等他开口——等他告诉她那条信息的内容。等她主动说，她没有。

她昨天在窗边等的也是这个。

"你以为我会说，"林延说，声音干涩。

"我以为你会说，"苏棠说。她停了一下，"你昨天在茶水间听到赵锐的话的时候，我就站在走廊拐角那边。"

这句话比扣分更重。林延觉得嗓子被人掐住了。

她知道。她从昨天就知道。她在等他主动开口，等了整整一天零一个上午。她给了他三次机会——会议室里，公告栏前，终端桌边。每一次她都留了门。

他每一次都选了沉默。

"我——"

"先别解释，"苏棠打断他。她的声音还是干净的，但干净里有了一层隔着东西的距离。"第二门还没结束，系统提示说还有一轮协调。这次我先选。"

她的手指已经在屏幕上了。她输入了一个选择，动作很轻，林延看不到她的屏幕。

终端屏幕上的红字消失了，换成了新的提示："等待对方输入。"

林延的手悬在键盘上方。他知道系统的逻辑——这一轮如果他选择和她对齐，能挽回上一轮扣掉的分。但如果她的选择是基于他对她的隐瞒做出的，那这个"对齐"本身就是不平等的。

他不知道她想让他选什么。他也不知道她输入的到底是什么。

赵锐在隔壁隔间里大概已经完成了他的选择。林延听见那边传来一声短促的蜂鸣，然后安静下来。

他输入了一个选择。闭着眼睛，凭直觉。

不是B线。不是A线。是C线——系统没有提示过的选项，需要手动输入编码，没有辅助信息，没有优先通道，什么都没有。纯粹的空白选择。

屏幕上跳出一行绿字："协调成功。路线不冲突，双方保留基础分。但C线为未优化路径，不计入最佳策略加分。"

扣分避免了。但最优解也丢失了。

苏棠看着屏幕，嘴唇抿成一条线。她没问林延为什么选C线，也没告诉他她选的是什么。

"先走吧，"她说，站起来，"还有第三门。"

她走出隔间的时候，林延看到了她的屏幕。在她转身的瞬间，屏幕上还残留着她输入的内容。

她选的是A线。

——那条需要辅助身份验证的优先通道。她输入的身份验证码是林延的编号。

她在替他选。在等他把灰色的那条信息共享给她之后，两个人一起走A线。

但他始终没有说。

走廊里的灯光比隔间里亮得多，照得林延的眼睛发涩。他站在原地，看着苏棠的背影在走廊尽头转了个弯，消失在楼梯口。

赵锐从隔壁隔间出来，路过他身边时脚步慢了一下。"第二门看上去不太顺，"赵锐说，声音里没有嘲弄，甚至不像挑衅，更像在陈述一个客观事实。"不过第三门才是决定性的。"

林延没回答。他还在想苏棠屏幕上的那行绿字，和她在窗台上敲的那两下手指。

一种成本正在成形。它不会在考核系统里显示为分数，不会出现在任何公告栏的名单上。但它真实存在，而且每过一门都在变得更大。
