# 第三章 不可逆的通过

第三门的考场不在楼里。沈主任的助理把他们带到了楼后的操场，跑道边上摆了四把折叠椅，椅子对面是沈主任本人。他坐在一张旧办公桌后面，桌上什么都没有，只有他自己的一双手。

赵锐第一个被叫上去。他站在办公桌前和沈主任说了大概五分钟，回来的时候脸上没有表情。接着是苏棠。她站起来的时候看了林延一眼——不是昨天那种等待的眼神，也不是今天上午那种隔着一层东西的疏远。是一种林延从未见过的审视，像在判断他接下来会怎么做。

苏棠在沈主任面前站了六分钟。回来的时候嘴角动了一下，幅度很小，林延读不出是轻松还是勉强。

然后叫到了林延。

"林延，"沈主任的声音不高，"第三门很简单。三门总分排名前两名的候选人可以获得晋升资格。目前赵锐排第一，苏棠排第二，你排第三。"

林延的手指在裤缝上攥了一下。

沈主任翻开面前的一页纸。"但是你有一个特殊条件——你在前两门的记录里表现出对合作方的信息保留。根据规则，存在信息保留记录的候选人需要在第三门做出一个公开选择。你面前有两个选项。"

他把纸推到林延面前。纸上只写了两行字。

第一行：放弃信息保留积分抵扣，承认保留行为，当前排名不变。赵锐第一，苏棠第二，林延第三。晋升资格按排名分配。

第二行：启用信息保留积分抵扣，将第一门和第二门的信息保留行为转化为战术选择加分，总分上调。上调后排名将变为：林延第一，苏棠第三，赵锐不变。

林延盯着那两行字，觉得自己的呼吸被什么东西压住了。

选第一行，苏棠晋升，他自己淘汰。选第二行，他晋升，苏棠淘汰。

这是一道二选一。没有中间路，没有C线可以手动输入。

"你有三分钟，"沈主任说，把桌面上的计时器翻了过来。秒针开始走动，声音在空旷的操场上格外清脆。

林延回头看了一眼苏棠。她坐在折叠椅上，抱着手臂，目光落在操场边的围墙上。她大概猜到了他在面对什么——她在里面站了六分钟，她可能也拿到了类似的选项。

不，等一下。

林延重新看向第二行。启用信息保留积分抵扣。如果苏棠在前两门没有任何信息保留——她确实没有，她从第一天就把所有材料推到了他面前——那她就没有"信息保留"可以抵扣。她的排名不可能因为这一项而变化。

但如果他的排名从第三跳到第一，被挤下去的只能是苏棠。

林延的喉咙发干。沈主任没有说话。秒针走了三十秒。

他想起了赵锐那句话："信不信由你。"如果那天下午赵锐没有在茶水间说那句话，如果他当时走进去了，如果他回会议室的第一秒就把那条消息告诉了苏棠——那么今天桌上这张纸可能根本不会出现。

但没有"如果"。

他又想起了苏棠在打印稿最后一页用铅笔写的那行轻飘飘的字："如果你有别的判断，告诉我。"她写的时候大概知道会被考核系统记录。她知道保留信息会触发第三门的惩罚。

她在提前给他留退路。

而他始终没有开口。

"还有两分钟，"沈主任说。

林延把手伸到纸上，手指停在第二行上方。他闭上眼睛，然后睁开。

"我选第一行。"

沈主任看着他，没有立刻动笔。"你确定？"

"确定。"

"理由？"

"没有理由，"林延说。然后他补充了一句，声音不大但很稳："如果非要说理由，那就是我不想用她留给我的退路来踩她。"

沈主任点了点头，在纸上写了几个字。他把纸收回去，合上面前的文件夹。

"第三门考核结束。最终排名：赵锐第一，苏棠第二，林延第三。根据规则，排名前两名的候选人——"

"等等。"

打断的是苏棠。

她从折叠椅上站起来，走到办公桌前。风吹过操场，把她额前的一缕头发吹散。

"沈主任，"她说，声音比任何时候都平静，"第一门的信息筛选环节中，我向林延传递了一份不合规的材料——我在打印稿最后一页留下了一条诱导性批注。这不属于正式通道的信息交换。请求扣除我第一门的战术加分。"

林延愣住了。打印稿最后一页——"如果你有别的判断，告诉我。"她把这句话算作不合规信息交换，用自我申报的方式，抵扣掉了他为她保留的排名。

沈主任沉默了五秒钟。然后他重新翻开文件夹，在苏棠的名下加了一行字。

"第一名，赵锐。第二名，林延。第三名，苏棠。前两名获得晋升资格。"

林延没有动。他还在看着苏棠。她的侧脸在操场的阳光下显得轮廓分明，没有表情，没有眼泪，没有愤怒，没有释然。什么都没多余。

她替他保住了他放弃的东西。但她用的是他自己的方式——保留信息，在最后一刻才说。

他选第一行放弃了自己。她用一个不完整的信息申报替他修正了结果。两个人都付出了代价：他失去了"自己承担后果"的干净，她失去了晋升资格。

而且他知道，她申报的那条信息其实并不违规。她在撒谎。或者说，她在承担一个不属于她的责任。

"林延。"苏棠叫他。

他抬起头。

"下次如果有什么话，"她说，"说完。说完再做选择。"

然后她转身走了。她走过跑道，走过赵锐的折叠椅，走回教学楼。她走的时候没有回头。操场上的风把她的头发吹得扬起又落下。

沈主任把最终名单递给助理。林延接过了他的那一份，纸张在手里轻得像没有重量。上面写着他的名字，第二名，通过。但通过这个词在这一刻失去了它本来的意思。

赵锐从他身边经过，说了一句："恭喜。"

林延没答。他走到苏棠刚才坐的那把折叠椅前，弯下腰，从地上捡起了一个东西——她起身时掉落的铅笔。笔杆上有她用拇指磨出的痕迹，细长而光滑。

他握着那支笔站在操场上，日光把他的影子拉得很长。

晋升名单会在明天贴出。他会在名单上。苏棠不会。

没人知道第三门真正发生了什么。沈主任的记录里只会有分数和排名，不会有操场上那两行字，不会有苏棠口中的那条"不合规信息"，不会有林延在纸张上方悬了三分钟的手指。

但他自己会记得。苏棠也会记得。

晋升不是终点。他心里非常清楚——下一段路，他欠的已经不只是一条没说出口的消息了。

他折好手里的通知，把它和那支铅笔一起放进口袋。口袋比昨天沉得多。

操场尽头的晚霞烧成一片橙红色。沈主任的助理搬走了四把折叠椅，办公桌也收走了。空旷的跑道上只剩下林延一个人。

他没有立刻走。他站在刚才苏棠站过的位置，面朝同一个方向。围墙上有一处掉漆的水泥痕，像一道很浅很浅的裂缝。

他看了一会儿那裂缝，然后把口袋里的铅笔转了半个圈。

明天开始，新的路。但不是从零开始的。是带着一条已经裂开的缝开始的。
