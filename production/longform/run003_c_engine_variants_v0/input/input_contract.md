# Input Contract

All four variants must use the same input files in this directory.

## Required Same-Source Fields

- world
- characters
- assessment mechanism
- three-chapter tasks
- initial relationship debt
- initial information gap
- reader question
- word range
- forbidden list

## Forbidden Input Divergence

Hermes must stop if any variant changes:

- protagonist, ally, competitor, or assessor
- world mechanism
- gate order
- chapter objective
- initial relationship debt
- initial signal truth status
- word range
- ending requirement
- forbidden list

## Permitted Difference

The only permitted difference is the C Engine method:

- C0: current C anchor
- C1: private deep fields
- C2: exemplar distillation
- C3: hybrid harness
