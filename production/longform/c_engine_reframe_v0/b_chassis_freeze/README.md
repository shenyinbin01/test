# B Chassis Freeze

B_longform_structure is frozen as the Longform Chassis candidate. Its job is to keep the book structurally coherent, not to create human-feeling prose.

The chassis preserves:

- world slice
- book spine
- volume card
- chapter cards
- state_delta proposal flow
- ledger preview
- reader question ledger
- consequence ledger
- continuity inheritance
- drift detection
- renderer hard boundary
- status and provenance discipline

B is chassis, not engine.
