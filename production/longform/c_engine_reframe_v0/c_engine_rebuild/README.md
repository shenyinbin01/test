# C Engine Rebuild

C is rebuilt from “more visible engineering controls” into a private Authorial Cognition Harness.

The rebuilt C Engine should model:

- author intent
- character pressure
- relationship debt as felt leverage
- scene heat
- reader decompression
- narrative stance
- anti-technique visibility
- renderer boundary v2

The foreground Writer/Renderer receives a natural-language scene brief, stance brief, safe continuity summary, and at most 1-3 rights-clear craft exemplars. It does not receive raw state deltas, full ledgers, raw scene pressure packets, Orchestrator notes, Reviewer scores, or operational status terms.
