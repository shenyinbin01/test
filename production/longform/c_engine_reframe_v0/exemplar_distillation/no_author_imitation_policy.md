# No Author Imitation Policy

This is not a specific-author imitation system.

Allowed learning targets:

- abstract mechanisms
- technique differences
- reader experience
- good/bad contrast
- rights-clear self-built samples
- public-domain craft abstraction

Forbidden targets:

- a living or specific author as a style goal
- copying a protected work's expression
- using author names as prompt anchors
- recreating signature diction or cadence
- saving protected source passages

The engine may learn that a scene needs subtext, pressure, delayed inference, and consequence. It must not learn “write like this named person.”
