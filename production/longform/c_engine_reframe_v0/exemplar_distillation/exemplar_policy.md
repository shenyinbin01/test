# Exemplar Policy

## Purpose

Exemplars help the engine learn transferable craft differences. They are not style targets and not source material for copying.

## Allowed Sources

- Self-built examples.
- Commissioned examples with clear rights.
- Rights-clear examples.
- Abstract craft features from public domain material.
- Good/bad brief pairs created internally.
- Internal synthetic examples created for a specific craft axis.

## Forbidden Sources And Uses

- Saving copyrighted raw corpus.
- Imitating a specific author.
- Copying a specific work's expression.
- Storing long original paragraphs from protected works.
- Using an author name as a style target.
- Using “like X author” as a goal.
- Treating surface diction as the transferable asset.

## Transfer Rule

Only transfer the craft principle, such as “reader infers a lie before the focal character admits it” or “subtext is carried by a rejected tactic.” Do not transfer names, sentences, scene objects, metaphors, cadence, or signature style.
