# Exemplar Distillation

Exemplars are used for technique contrast and distillation, not author imitation.

The goal is to teach differences such as:

- thick scene vs thin scene
- hot scene vs flat scene
- subtext vs manual explanation
- character can-only-choose-this vs character executes task
- reader decompression vs report-style summary
- narrative stance vs generic style command

All exemplars must be rights-clear, synthetic, commissioned, public-domain-derived at the abstract technique level, or self-built. No raw copyrighted corpus is stored in this package.
