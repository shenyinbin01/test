# Copyright Boundary

This package does not use raw copyrighted corpus.

Rules:

- Do not save copyrighted raw corpus.
- Do not commit long source passages from protected works.
- Do not target a named authorial style.
- Do not use a protected work as a style prompt.
- Do not create an imitation task from author name, book name, or signature surface.
- Use synthetic, rights-clear, commissioned, self-built, or public-domain abstracted samples only.

If a craft insight comes from reading external fiction, store only the abstract principle, never the expressive passage.
