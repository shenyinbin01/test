# C2 Exemplar Distillation

C2 tests technique exemplars without changing the core C anchor as deeply as C1.

## Enabled

- B chassis
- safe C anchor
- 1-3 rights-clear good/bad craft exemplars per target scene type
- exemplar policy
- anti-technique visibility review

## Exemplar Types

- hot vs flat scene
- subtext vs explanation
- character pressure vs task execution
- reader decompression vs report summary
- narrative stance vs generic surface style

## Hypothesis

Good/bad contrast helps the renderer avoid flat process prose. The risk is that exemplars may become surface imitation or decorative prose if not tightly governed.
