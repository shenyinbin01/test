# Run 003 Experiment Design

## Design Shift

run_003 is no longer an equal ABC comparison.

- A is historical anchor only.
- B is frozen chassis.
- C variants compete to prove prose quality, not report completeness.

## Goal

The goal is not a better report. The goal is prose that:

- reads more like fiction and less like process,
- makes characters feel alive,
- makes scenes hotter,
- leaves room for reader decompression,
- preserves webnovel propulsion,
- makes readers want the next chapter.

## Variants

| Variant | Definition | Purpose |
|---|---|---|
| C0 | Current C anchor | Baseline for current scene_agency_packet/orchestrator style |
| C1 | Private deep fields | Test whether deeper private pressure fields improve character life without field leakage |
| C2 | Exemplar distillation | Test whether good/bad technique samples improve rendering without author imitation |
| C3 | Hybrid harness | Combine private pressure modeling, reader decompression, stance control, and exemplar distillation |

## Evaluation

Main evaluation is human blind pairwise preference. LLM reviewer only checks for visible technique, leakage, weak pressure, and obvious failure modes.
