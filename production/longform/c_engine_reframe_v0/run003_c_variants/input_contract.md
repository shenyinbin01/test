# Run 003 Input Contract

All C variants must use the same inputs.

## Shared Inputs

- Same `world_slice`.
- Same `book_spine`.
- Same `volume_card`.
- Same three-scene or three-chapter brief.
- Same character set.
- Same initial relationship debt.
- Same initial information gap.
- Same reader question seed.
- Same word range.
- Same forbidden list.
- Same B chassis constraints.

## Forbidden Input Differences

Variants must not differ by:

- easier plot assignment,
- different stakes,
- different cast,
- different ending requirement,
- different hidden information,
- different word budget,
- different accepted state,
- different official ledger.

## Shared Purpose

The only intended difference is how C turns private structure into readable prose.
