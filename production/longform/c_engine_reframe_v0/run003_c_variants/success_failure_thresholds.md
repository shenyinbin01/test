# Success And Failure Thresholds

## C3 vs C0 Success Thresholds

C3 must show:

- reader_pull preference >= 65%
- anti_ai_surface preference >= 65%
- character_alive preference >= 60%
- scene_heat preference >= 60%
- technique visibility complaints down >= 30%

## Failure Criteria

Fail or pause C3 if:

- C3 is more complex but harder to read.
- C1 or C3 leak private fields.
- Reviewer scores are high but humans do not prefer it.
- C2 improves only surface style.
- C3 performs worse than C0.
- C3 sacrifices webnovel propulsion for literary fog.
- Text feels managed and lifeless.
- Reader decompression is explained away.
- Scene heat becomes loud confrontation without tactic turn.
- Character desire is only a label.

## Decision Rule

Run 003 does not accept C3 because it is more engineered. It accepts C3 only if blind readers prefer it as fiction while B chassis discipline remains intact.
