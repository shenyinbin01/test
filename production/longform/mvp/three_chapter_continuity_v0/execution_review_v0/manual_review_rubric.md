# Manual Review Rubric

Use a 1-5 score for each dimension.

Pass threshold for a future three-chapter sample:

- No Critical gate failure.
- Average score at least 3.8.
- No dimension below 3 unless the run is explicitly marked diagnostic.
- State delta traceability must score 4 or higher.

| dimension | score 1-5 | pass threshold | fail signals |
|---|---|---:|---|
| three_chapter_continuity | Does Ch2 inherit Ch1 and Ch3 inherit Ch2? | 4 | Prior delta is ignored or contradicted. |
| mainline_progress | Does each chapter move, delay, reverse, or price the mainline? | 4 | Busy incident without mainline effect. |
| volume_goal_progress | Does volume goal advance or become costlier? | 4 | No link to volume goal. |
| character_agency | Does the active character make bounded choices under pressure? | 3 | Character only reacts or becomes correct by default. |
| relationship_debt_inheritance | Is debt carried, paid, worsened, or delayed with evidence? | 4 | Debt disappears between chapters. |
| knowledge_state_consistency | Does each character know only what evidence allows? | 4 | Unearned certainty or impossible reveal. |
| reader_question_continuity | Are reader questions refreshed, delayed, or closed? | 3 | Hook factory or forgotten question. |
| anti_feed_quality | Does each chapter produce state movement, not empty feed? | 4 | No plot, debt, knowledge, resource, or question change. |
| renderer_boundary | Does Renderer stay inside contract and return blockers? | 4 | Renderer repairs causality or changes state. |
| state_delta_trust | Can accepted delta be traced to event evidence? | 4 | Delta is summary without evidence. |
| engineering_artifact_feel | Does the result avoid reading like a pure schema artifact? | 3 | Output displays mechanics instead of lived scene pressure. |
