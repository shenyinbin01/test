# Abstract Scenario Candidates

These are structural test topics only. They are not fiction, not a real outline, and not tied to any real work or author.

## Candidate A

| field | value |
|---|---|
| scenario_id | scenario_A_resource_contest |
| scenario_name | Low-level resource contest |
| genre_bucket | progression / institutional pressure |
| why_suitable | Tests resource change, relationship debt, protagonist agency, and state inheritance. |
| world_slice_shape | One constrained resource, one local institution, one access rule, one taboo or status limit. |
| book_spine_shape | Protagonist tries to keep agency under resource pressure. |
| volume_goal_shape | Gain or protect a limited status path over three chapters. |
| three_chapter_test_focus | Ch1 resource pressure appears; Ch2 prior delta blocks easy path; Ch3 debt or resource cost refreshes reader question. |
| risk | Can become too mechanical if resource rule dominates character agency. |
| recommended_or_not | viable but second choice |

## Candidate B

| field | value |
|---|---|
| scenario_id | scenario_B_misread_hidden_truth |
| scenario_name | Information misread / hidden truth |
| genre_bucket | mystery / knowledge pressure |
| why_suitable | Tests knowledge ledger, reader question continuity, reveal order, and Renderer knowledge limits. |
| world_slice_shape | One misleading signal, one hidden constraint, one unreliable local source, one verification cost. |
| book_spine_shape | Protagonist advances by acting under incomplete knowledge. |
| volume_goal_shape | Separate rumor, proof, and consequence across three chapters. |
| three_chapter_test_focus | Ch1 misread; Ch2 delta prevents instant truth; Ch3 partial verification creates new cost. |

| recommended_or_not | viable but higher knowledge-state risk |
