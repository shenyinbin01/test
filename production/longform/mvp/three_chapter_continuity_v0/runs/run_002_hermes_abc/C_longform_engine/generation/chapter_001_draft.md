# 第一章：第一个信号

林砚收到通知的时候，窗外正下着入秋以来的第三场雨。

通知是沈组长群发的——考核窗口开放了。三关，依次推进，资源有限制，每关只能用一个主力策略。

林砚把通知看了两遍。第一遍看规则，第二遍在找自己名字没有被单独划出的证据。没有。这意味着他和所有人一样，站在同一条起跑线上。

但苏棠的脸色告诉他，事情没那么简单。

"你的准入凭证呢？"苏棠从对面的卡座探过头，声音压得很低，"赵锐说他已经领了。"

林砚的手指在桌面上敲了两下。准入凭证——进入考核流程的资格令牌，申请窗口开了三天，今天是最后一天。他还没申请。

不是忘了。

是他收到了一条「温馨提示」：有人告诉他，第一关的题目会在正式考核前以私下途径流出，如果他想「提前准备」，可以走某个非公开的渠道。

这条消息没署名。林砚追问过一次，对方只回了一句"你懂得"。

"我在想别的事。"林砚说。

"什么事比准入凭证还重要？"苏棠的语气里有一点不耐烦，但更多的是担心。她比林砚早半年进组，这次考核她需要搭档分担压力。林砚知道她指望着自己。

林砚看了看时间。距离准入凭证的申请截止还有四个小时。

他面临一个选择。

选择 A：立刻提交申请，拿下准入凭证——但这样一来，他就没法在不留下记录的情况下确认那条"温馨提示"的真实性。那个非公开渠道只能用一次，而且一旦申请了正式凭证，对方就不会再理他。

选择 B：用仅剩的这段时间去接触非公开渠道，先搞清楚第一关的真正内容——但如果在截止前没来得及折回来申请凭证，他就连入场资格都没了。

"林砚。"苏棠叫了他的全名，这不是她的习惯。她认真的时候才会这样。"你在犹豫什么？"

林砚看着她，看见她眼睛里的紧张。

他忽然意识到，苏棠把自己的准备押在了他身上。她需要他过了第一关，才能在第二关给她提供信息。这不是他一个人的事。

"我知道了。"林砚说。

他打开了申请页面，填了名字，提交。

系统弹出一行字：「准入凭证提交成功，当前排队人数：7，等待时间：约三小时。」

三小时。正好卡在截止时间线上。

林砚合上屏幕，对苏棠说："我去冲杯水。"

他在茶水间站了三分钟，把那条"温馨提示"从手机里删掉了。

他不知道那条提示的真假。可能是赵锐放的烟雾弹，想让他错过申请窗口；也可能是真的，有人想借他当探路石。不管是哪种可能，他选择相信规则门槛比小道消息更重要。

但他也清楚，这个选择的代价是——他进入第一关的时候，对题目的方向一无所知。

而他删掉消息的那一刻，苏棠不知道发生了什么。她只知道林砚提交了申请，以为万事妥当。

这就是第一笔债。

---

沈组长在下午六点公布了考核进度：「准入凭证发放完毕。第一关：信息甄别，明早九点开始。请所有入围者自行准备。」

林砚看了一眼名单。八个人入围。赵锐排在第一个。

赵锐朝他笑了笑，那种笑容不像敌意，更像是一个准备好了的人看着一个还没睡醒的人。

林砚回了苏棠一条消息："明早见。"

苏棠的回复很快："你紧张吗？"

"还好。"

他没说实话。他紧张的不是考核本身，而是那条已经被删掉的提示内容——如果那个信号是真的，那他主动放弃了一次提前了解考题的机会。

但如果信号是假的，那他从一开始就在往坑里跳。

哪一种？他没有办法现在验证。

他只知道自己做了一次选择，而且留下了痕迹。
