# 第001章：第一个检查点——两难之选

林远盯着电脑屏幕上那封落款「人力资源部——陈主任」的邮件，手心微微出汗。

邮件内容很简短，但每个字都重若千钧：「年度高级项目经理晋升考核将于下周一正式启动。本次考核设三个检查点，采取三方评估制——项目成果占40%、同事互评占30%、上级评价占30%。仅一个名额。请各位候选人做好准备。」

一个名额。

林远的目光在那个数字上停了好几秒。整个项目部有三个符合条件的骨干，但所有人都知道真正的竞争只会在他和赵铭之间。老周年纪大了，早就放弃了往上爬的心思，只是被系统自动纳入了候选名单。

他需要信息。需要比别人更多的信息。

林远拿起水杯，借着去茶水间的机会经过赵铭的工位。赵铭不在。座位上摊着几份打印出来的文件，标题栏隐约能看到「考核评分细则」几个字。林远心头一跳——赵铭已经弄到了内部资料。

「找陈主任要的？」林远回到工位后，给陈主任发了条消息。

十分钟后，陈主任的回复到了：「评分细则会在周一统一公布。提前打听没有意义。」

标准回答。但也等于确认了——赵铭没有经过正规渠道拿到那份东西。至于他是从哪里弄到的，林远猜得到：赵铭的舅舅是公司总部的人事副总监。

林远深吸一口气。先不管赵铭的信息优势，他手上有自己的筹码。上个月他主导的「东区供应链优化」项目刚刚结项，客户评分很高。按照考核规则，他应该能争取到「项目优先调配权」——这意味着在第一个检查点，他可以优先选择一个高价值的新项目作为自己的考核项目。这是实打实的资源。

他拨通了项目调度组的内线电话。

「喂？王哥，我是林远。下周那个新开的客户项目，我想——」

话没说完，手机震动了一下。屏幕上弹出一条微信消息。

苏晚：「林远，方便说话吗？有个事情想请你帮忙。」

林远的手指停在电话按键上方。

苏晚。三个月前，他们还是同一个项目组的搭档。那时候苏晚帮他扛过一个大雷——客户投诉到总部，苏晚主动揽下责任，替林远保住了绩效评分。之后苏晚调去了后勤部，两人联系渐渐少了，但林远心里一直记着这笔账。

「王哥，稍等一下。」林远放下电话，给苏晚回了消息。

苏晚的语音带着一丝犹豫：「你最近是不是在做那个晋升考核？」

「对。」

「我这边有个麻烦。之前咱们那个东区项目的遗留文档，后勤部在查，涉及到几个签字的流程问题。如果上面查下来……可能会影响你那个项目的结项评价。」

林远的心沉了一下。

苏晚继续说：「陈主任那边要求补交一份材料，需要用到你手上的项目完整档案。但档案的调用权限在你手里——是你当时申请的优先调配权绑定的。你能不能……」

林远明白了。那份「项目优先调配权」不只是可以用来选新项目，它的法律意义是绑定了整个项目档案的管理权限。如果他把这个资源用在帮苏晚调档案上，就等于放弃用它去争取新项目。

电话那头，王哥还在等。

林远闭上眼睛。三秒后，他做了决定。

「王哥，新项目的事先不聊了。」他挂掉内线，给苏晚发了条消息：「档案权限我下午给你开通。需要签字的文件你发过来。」

消息发出去的那个瞬间，林远感到一种奇异的轻松——像是终于卸下了什么。但也同时感到一种更深的焦虑：第一个检查点的资源项，他已经输了。

下午三点，第一个检查点的材料提交截止。

陈主任在会议室里逐个点开候选人的提交材料，表情一如既往地淡然。投影仪上，林远看到自己的资源项旁边被打了个黄色的标记——「待加强」。

坐在对面的赵铭微微侧过头，嘴角带着一个不易察觉的弧度。

林远低下头，在笔记本上写了一行字：

「人情还了。后面两个检查点，必须赢。」

但他不知道的是，苏晚的事情才刚刚开始——而他欠下的这笔「换回来的关系」，会在第二个检查点以一种他完全没想到的方式重新找上门来。
