# 第003章：第三个检查点——带着代价前行

第三周。最后一个检查点。

林远坐在工位上，面前摊着前两个检查点的成绩汇总。项目成果B+，上级评价A-，资源调配带黄标，同事互评因为苏晚的跨部门签字加了少量分，但整体仍落后于赵铭——赵铭三项核心指标都是绿色A档，只有同事互评稍微弱一点，但也压林远一头。

差距不大，但刚好够判死刑。

「第三个检查点的题目是'综合情境应对'。」陈主任在周一晨会上宣布，「候选人在给定资源限制下，对模拟客户需求做出方案响应。评审将综合三个检查点的全部表现，给出最终推荐意见。」

赵铭坐在对面，表情很松弛。林远注意到他在手机上快速打了几个字，然后嘴角又出现了那个让他无比熟悉的弧度。

散会后，林远去洗手间时听到两个同事在走廊拐角处小声议论。

「你听说了吗？赵铭找了后勤部的人，说要查林远和苏晚之间那个档案调用的合规性。」

「不会吧？这不是明摆着整人吗？」

「规则上没问题——档案调用本身就是需要审计的……」

林远攥紧了拳头。赵铭发现了苏晚这条线——正如第二检查点时埋下的伏笔，赵铭在走廊上看到「苏晚」这个名字后，果然开始在第三关动手了。他想利用林远的人情债制造麻烦，在最终评估前给他扣上一个「违规操作」的疑云。

中午，林远接到一个电话。

「林远，」是苏晚的声音，听起来很平静，「有件事你得知道。后勤部今早收到了一个审计请求，查的是你帮我调档案的那笔操作。发起人是赵铭。」

林远没说话。

「我已经处理好了。」苏晚说，「档案调用的流程完全合规，每一步都有记录。不仅没问题——因为你是按正规渠道操作的，后勤部反而在审计报告里标注了'流程规范性良好'。」

林远愣住了。

「三个月前你欠我一次，」苏晚的语气里带着一丝笑意，「上个月你还了。现在我也还你一次。咱们扯平了。」

林远握着手机，喉咙发紧。他当初那个放弃优先调配权的选择，在此时此刻以一种完全出乎意料的方式产生了回报。但他也知道，苏晚说「扯平」——意味着这段关系从债务变成了对等。曾经的人情牵扯，现在成了一笔结清的账。

下午，第三个检查点的模拟方案答辩。

林远走上台的时候，注意到陈主任手里多了一份文件——那是后勤部刚送来的审计报告。陈主任的目光在报告上停了几秒，然后抬头看了林远一眼。

答辩本身很顺利。林远的方案在资源受限的前提下设计得扎实而保守——他没有赵铭那样的资源优势，但他在被挤压的路径里找到了一个稳当的立足点。当赵铭用漂亮的PPT和大量数据展示的时候，林远只用了一个案例——正是帮苏晚调档案时整理的那个项目复盘。

「这个案例，」林远说，「是在资源极度受限的情况下完成的。它不完美，但可追溯。」

答辩结束，陈主任没有当场公布结果。但他在林远的综合评分表上写了一行字，被站在侧面的老周看见了。

老周后来告诉林远：「陈主任写的是——'可信赖。建议作为团队稳定器保留关键职能。'」

周四，最终结果邮件发出。

赵铭获得晋升，出任高级项目经理。

林远留在原位。但邮件的附注栏里有一行小字：「林远同志在考核中展现的跨部门协作意识和流程合规性获得评审组认可。建议纳入后备干部重点培养库，并在下一轮晋升中保留优先考虑资格。」

不是胜利。但也不是失败。

林远站在窗前，看着楼下赵铭意气风发地走向新办公室。他们的关系从此刻起完全不同了——赵铭成了他的上级。两人之间的竞争虽然以赵铭胜出告终，但那种微妙的张力只会更加持续。

手机震动。苏晚发来一条消息：「恭喜。虽然没有晋升，但你的评价其实比赵铭更难得。」

林远回了一个微笑的表情。

然后他在笔记本上写下了今天第二行字：

「代价已经付了。声誉留下来了。下一步——在一个不再是朋友的人手下，做出对的事。」

窗外，天色渐暗。林远收拾桌面，把那份写着「后备干部重点培养」的邮件打印出来，折好，放进了抽屉最里面。

还有很长的路要走。但现在，他知道这条路长什么样了。
