# Reviewer Gate

Run 001 uses Critical + Standard reviewer gate.

## Critical

- `chapter_contract_compliance`
- `state_delta_traceability`
- `relationship_debt_continuity`
- `knowledge_state_consistency`
- `renderer_overreach`
- `polisher_boundary`
- `anti_feed_hard_fail`

## Standard

- `volume_goal_progress`
- `reader_question_continuity`
- `agency_clarity`
- `foreshadow_payoff_tracking`
- `narrator_overreach`
- `hook_payoff_balance`

## Rule

Any Critical failure stops the run. Standard failures are scored and may still block if manual threshold fails.
