# Risk And Failure Modes

| failure_mode | cause | detection | mitigation | owner_layer |
|---|---|---|---|---|
| Work Voice is mistaken for ordinary style polishing | Packet overemphasizes sentence rhythm and tone labels | Reviewer sees no narrator_position effect | Require stance relation fields in every brief | Work Voice Contract |
| Work Voice is mistaken for author imitation | Source identity leaks into runtime prompt | Contamination checklist fails | Ban source work as style label and author target | Contract / Reviewer |
| Runtime packet is too long and makes Writer stiff | Too many upstream observations are injected | Writer output reads like checklist | Compile compact packet only; no observation cards | Contract compiler |
| Planner outputs stance fields but Writer explains them | Writer treats fields as content | Draft contains meta-explanation or report-like narration | Writer rule: fields guide prose, never appear in prose | Writer |
| Reviewer checks field presence only | Gate schema becomes mechanical | Text has fields but no stable storyteller | Reviewer must check body-level effect and return layer | Reviewer |
| Polisher overreaches and adds stance | Reviewer routes structural failure to Polisher | `polisher_allowed` true despite stance failure | Add no_polisher_overreach hard gate | Reviewer / Polisher |
| Human Texture and Work Voice overwrite each other | Focus fields and stance rules are not coordinated | Character has details but no voice, or voice but no people | Planner pairs fields per scene; Reviewer separates failure cause | Planner / Reviewer |
| C group becomes stiff, old-fashioned, or hard | Contract overvalues solemn stance | Reviewer scores momentum / tone failure | Add webnovel_momentum_preserved gate | Writer / Reviewer |
| C group loses webnovel propulsion | Voice rules slow conflict and payoff | Low payoff_visibility / momentum score | Keep beat priority and payoff timing | Planner / Writer |
| Anti-contamination makes voice too abstract | All concrete detail is removed from contract | Writer cannot execute brief | Keep abstract rules tied to scene_type and detail_bias | Contract |
