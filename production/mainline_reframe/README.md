# Mainline Reframe

本目录记录 Deep Research 第一轮之后的主线优先级调整。

核心变化是：当前不再优先继续扩展 Work Voice，而是先造“发动机”。这里的发动机不是一个新 agent，而是角色主动感 / 行动主权（Character Agency）和后果继承账本（Consequence Ledger）的前置设计。

本目录只做路线重排和工程规划，不修改 `skill-pack`，不生成正文，不运行验证链路。
