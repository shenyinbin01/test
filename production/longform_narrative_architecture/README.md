# Long-form Narrative Production System

长篇叙事生产系统解决的不是“单章怎么写得像人”，而是“如何从 IP 宇宙稳定生产一部长篇小说”。

它可以理解为车架、传动、导航和状态系统。发动机是角色主动感 / 行动主权（Character Agency）和后果继承账本（Consequence Ledger），车架不能替代发动机，但能让发动机的输出进入卷、章、场景和下一章状态。

本目录只做框架设计和第二轮 Deep Research 输入，不修改生产链路。
