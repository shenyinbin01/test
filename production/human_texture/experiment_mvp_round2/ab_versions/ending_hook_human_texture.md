# Ending Hook Human Texture Version

说明：实验性 B 版，不替换正式正文。剧情功能保持：白签、关系余波、天幕裂缝、元启降临、枯竭提前、标记、回收标准。

```text
执事宣布继续复检。

柳青砚回到正式弟子的队列里。苏衍走过去拿签。

白色的。

签很薄，被他攥在手里，边角立刻折出一道痕。

这不是通过。只是执事把问题往后推了一步。

队伍继续往前。苏衍没有回杂役队列。他站在演武场边的枯松下，看着柳青砚的背影。

他想走过去。

胖少年在正式弟子队列里半抬了一下手，像是想问自己刚才那句话算不算惹祸。苏衍也没回应。

他现在只要回一个眼神，就会多欠一笔。

柳青砚没有回头。

苏衍在心里把能说的话过了一遍。谢谢。不关你的事。我有苦衷。哪一句都像新的谎。

复检结束。

黑衣人在竹简上写字。笔尖停了一下。

其中一个抬头看天。

苏衍本能地也抬头。抬到一半，他忽然想起自己刚才还想叫柳青砚。

晚了一息。

天空裂开了。

演武场上所有人都仰起头。金色裂缝横在云边，冷得像把天幕切开。苏衍经络里的积分同时震了一下，本来的四十五点、回收来的、烧剩下的，全都像被一只手捏住。

他下意识去看柳青砚。

柳青砚也感觉到了。她的手已经摸到剑柄，却没有拔剑。她看向天空，没有看他。

执事攥紧竹简，指节发白。嘴唇动了一下。

“三年。——他说还要三年的。”

然后那个人从裂缝里降下来。

不是走，是降。地面被震得裂开，青石板从演武场中央往四周裂出蛛网纹。尘土散开后，一个穿灰袍的男人站在那里，袍边的金色符文缓慢流动。

他笑着，看向整座青云宗。

“我叫元启。”

声音不大，却压住了所有人。

“受古阵之命，对青云宗进行积分回收。”

没人问。有人张了张嘴，又闭上。每个人都在等别人先把那个最怕的问题说出来。

执事先开口。

“你早了。按周期算，最少还有三年。”

元启看了他一眼。

“周期变了。有人在过去三天里抽走了阵眼里最后一点活性积分。枯竭点提前了。”

执事的脸白了一瞬。

苏衍知道执事知道。

执事没有看苏衍。

这种不看，比看更重。

元启举起手。掌心亮起圆形符文。演武场上所有人的积分同时闪了一下。

不是检测。

是标记。

温暖只停了一瞬，随后恐惧落下来。胖少年往后退了半步，撞到身后的人。柳青砚终于转头，目光掠过苏衍，又停在元启身上。

她没有再给他台阶。

“回收标准很简单。”元启说，“积分纯度。纯度高，保留。纯度不足——”

他笑了笑。

“回收。”

那天晚上，苏衍坐在杂役宿舍门槛上。

天上的裂缝还在，比白天更低。屋里有人压着声音哭，有人翻身装睡。胖少年没有来找他。柳青砚也没有。

白签被他放在膝盖上，折痕还在。

他听见碎石路上传来脚步声，抬头时只看见柳青砚的背影。她停过，但没有停到让他开口。

苏衍张了张嘴。

没有叫出声。

天空的裂缝又亮了一下。

这一次他先看见的不是裂缝。

是柳青砚没有回头。
```
