# Ending Hook Baseline

来源：`production/phase8/skill_injection_minimal/validation_5ch/drafts/chapter_004_draft.md`

```text
执事宣布继续复检。柳青砚回到了正式弟子的队列里。苏衍走过去拿他的签。

白色的。

但苏衍知道那不是通过。那只是执事的沉默——一种把问题推到更晚、更不可能解决的时间的沉默。

队伍继续往前移动。苏衍没有回杂役队列。他站在演武场边的一棵枯松下面，看着柳青砚的背影。他想走过去说点什么，但他想不出来有什么话能让一个被你面对面撒谎的人重新相信你。

复检结束了。

站在执事身后的那两个黑衣人在竹简上写了什么。其中一个人抬起头，朝天空看了一眼——苏衍顺着他的目光看过去。

然后他也看到了。

云的边缘，极远处的高空，出现了一条裂缝。不是那种云被风吹散的缝隙。是一条笔直的、像被刀子划开的裂缝，沿着东西方向横贯了半边天空。裂缝的边缘在发光——一种比太阳更金黄、比火更冷的金色。

演武场上的所有人都抬起了头。有人发出了小声的惊叫。

然后苏衍感觉到了——不是看见，是感觉到——他的经络里所有的积分，无论是他本来的那四十五点，还是他回收来的，都在同一时间震颤了一下。像有人在他的血管里拉了一根弦。

执事也感觉到了。苏衍看见他攥紧了手里的竹简，指节发白。他仰着头看那道金色裂缝，嘴唇无声地动了一下。

苏衍读懂了他的口型。

"三年。——他说还要三年的。"

然后苏衍看见了从裂缝里走出来的人。

不是走。是降。像一截断崖从天空坠落，砸在演武场的中心。地面剧烈地震动了一下，青石板裂成了蛛网状的纹路。

尘土散去之后，一个男人站在裂缝中间。他穿着一身看不出材质的灰袍——不是宗门的长老袍，不是官府的官服，是一种苏衍从未见过的织物，袍边上金色的符文像活物一样在缓慢流动。

他看起来很年轻，大概三十岁出头。他是笑着的。不是友善的笑，也不是恶意的笑。是一种"这里果然还在用这套系统"的了然的笑。

"我叫元启。"

他的声音不大，但每一个字都清清楚楚，好像不是从耳朵里进来的，是直接印在了听者的意识里。

"受古阵之命，对青云宗进行积分回收。"

全场没有人说话。不是因为不想问——是因为没有人知道该问什么。积分回收是什么？怎么回收？谁的积分？所有人脑子里都塞满了问题，但所有人都在等别人先开口。

老执事先开了口——不是提问，是陈述。

"你早了。按周期算，最少还有三年。"

元启歪了歪头看他。那个表情就像成年人听到小孩说"天黑了就会有怪物"。

"周期变了。"他说。"枯竭加速了。有人在过去三天里抽走了阵眼里最后一点活性积分。枯竭点提前了。"

执事的脸白了一瞬间。他知道是谁抽走了活性积分。苏衍知道执事知道。

但执事没有看苏衍。

元启举起了一只手。他的掌心亮起了一个跟矿洞里阵眼一模一样的圆形符文图案。演武场上的所有人——正式弟子和杂役——经络里的积分同时闪烁了一下。不是检测。是标记。每一个人都感觉到了一瞬间的温暖——然后温暖消失了，取而代之的是恐惧。

"回收标准很简单。"元启说，"积分的纯度。纯度高——没有杂质的积分——保留。纯度不足——"

他对着所有人笑了笑。

"回收。"

那天晚上，苏衍坐在杂役宿舍的门槛上。他的积分在经络里安静地流淌——每一分都带着不属于他的情绪碎片。他抬头看天，那条裂缝还在那里，比白天的位置更低了一点，像一只慢慢睁开的眼睛。

他感觉有人在看他。不是元启。是柳青砚——站在远处的碎石路上，手里没有剑，站了很久。然后他转身走了。

没有质问。没有追问。连失望都没有了。

苏衍第一次感觉到，认知优势最大的代价不是吸收别人的情绪。是把所有知道真相的人一个一个推开。

天空的裂缝又亮了一下。
```
