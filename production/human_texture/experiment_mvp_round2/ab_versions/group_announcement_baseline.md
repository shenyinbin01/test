# Group Announcement Baseline

来源：`production/phase8/skill_injection_minimal/validation_5ch/drafts/chapter_005_draft.md`

```text
第二天清晨，元启在演武场中央竖起了一面古镜。

不是一般的镜子。镜面不是玻璃，是一种苏衍叫不出名字的深灰色材质，表面平滑得像液体，但摸上去是固体。镜子高三丈，宽一丈，立在演武场正中央，镜面对着宗门大殿的方向。

"你们想知道真相。"元启站在镜子旁边，一只手搭在镜框上。"想知道的站到镜前来。不想知道的可以走。不走的人——"

他笑了笑。

"不走的人等于默认想保留积分。保留积分就要接受检测。"

没有人走。

不是没人想走。是没有人敢第一个走。一个刚入门的杂役弟子想回头，被旁边的人拽住了袖子。

苏衍站在人群的最后面。他离镜子的距离大概二十步，但他不需要站在镜前，就已经能看到镜面上浮动的画面——那是跟他在矿洞阵眼里看到的同一种画面。历代弟子的积分分配记录，按年份排列，从几百年前一直排到现在。

"看清楚了吗？"元启的声音飘过来。

"你们的积分是从哪里来的？天地灵气？生来就有？不是。"

他用手指在镜面上划了一下。画面拉远，变成了一个巨大的阵法展示图——跟矿洞里的阵眼一模一样，但更大，覆盖了方圆数百里的山川水系。

"这是古阵。你们所有人——方圆三百里内所有人——的积分，都来自这个阵法。"

镜面上的画面继续流动。苏衍看到了自己——准确说，是看到了自己经络积分的变化曲线。从四十五起步，一路上涨到一百一十二，然后突然掉到了六十一。

有人在吸气。

元启的目光穿过人群，落在苏衍身上。他的嘴角弯了一下。

"唔，有意思。"

但元启没有追问苏衍。他把注意力转回镜子，继续展示下一组画面——这组画面让全场彻底安静了下来。

枯萎。

巨大的阵法图形从饱满的金色慢慢变成暗淡的灰黄。每过大约一百年，颜色就淡一层。每过一个周期，接收积分的弟子数量就少一批。

"三千年。"元启说。"古阵的设计寿命就是三千年。每一次枯竭都是一次筛选——那些能真正利用积分、理解系统规则的修士存活下来，成为修真界的中坚。那些只能被动接收积分的——"

他停了停。

"他们的积分会被回收，重新分配给下一次枯竭后诞生的新弟子。"

场面开始乱了。有人在哭，有人在骂，更多的人站在原地，脸上的表情像是刚发现自己活在一场考试里。
```
