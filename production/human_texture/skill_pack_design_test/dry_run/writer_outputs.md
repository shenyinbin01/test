# Writer Outputs

本文件记录 5 个片段基于压缩版 Human Texture packet 的 B 版短改写。

说明：这些 B 版只用于 dry run，不替换正式正文，不重写完整章节。

## 1. C4 柳青砚关系节点

### Baseline 摘要

苏衍复检读数六十一，柳青砚指出异常。胖少年替苏衍作证，执事用证人规则压住指控。柳青砚低声说自己是在给苏衍台阶，苏衍回答“我没有秘密”，柳青砚失望后退，苏衍拿到白签。

### Baseline 问题

- 苏衍撒谎像理性隐藏秘密，羞耻和自保没有变成动作。
- 柳青砚失望被命名，但关系债不够具体。
- 胖少年作证有效，但像功能性证人。
- 复检场制度压力存在，但没有充分改变人物选择。

### Focus Fields

- `shame_or_avoidance`
- `relationship_debt_change`
- `consequence_next_friction`

### B 版短改写

```text
执事看向苏衍时，白签和红签还摆在小桌上，签角朝外。

苏衍张了张嘴，准备说“可能是你感知错了”。话到舌尖，他忽然觉得那句话太轻，也太脏。

“但是他输了。”

胖少年从正式弟子队列里挤出声音，脸涨得通红。他替苏衍说完那几句辩解，立刻低下头，像刚把自己的白签也押了出去。

苏衍没有看他。他知道自己欠了这句话，可现在点一下头，都会把人拖进来。

执事宣布需要双方证人。杂役队列里有人把手缩进袖子，另一个人盯着地缝。没人站出来。

柳青砚往前两步，声音压到只有苏衍能听见。

“我是在给你台阶。你公开一点，就不用一个人扛。”

他停了一下。

“也不用把别人一起瞒进去。”

苏衍指尖冷了一下。他想到柳青砚掌心那道金线。承认秘密，也会把柳青砚的秘密带出来。

他很想说“不关你的事”。那至少像是保护。

但他说出口的是：“我没有秘密。”

柳青砚退了一步。不是生气，是把刚递出来的东西收回去。

白签到手时，苏衍手心的汗把签角浸软了一点。胖少年那句话、杂役队列的沉默、柳青砚退开的那一步，都留在他身后。
```

### 剧情功能是否保留

保留。复检通过、柳青砚指控、胖少年作证、证人规则、苏衍撒谎、关系破裂、白签到手均未改变。

### Dry Run 判断

压缩 packet 足够驱动关系节点。B 版不需要显性写“关系债”，读者能从不看胖少年、不接受柳青砚台阶、白签汗痕中读到后果。

## 2. C3 饭堂 / 矿洞信息露出节点

### Baseline 摘要

苏衍在饭堂听老杂役说后山矿洞曾是修炼室，随后顺着金线进入矿洞。他看到阵眼，直接理解这是积分分配系统，触碰阵眼后获得记忆洪流，得知枯竭不是事故而是周期筛选。

### Baseline 问题

- 饭堂有生活质感，但很快变成线索触发器。
- 主角“无需知识就能看懂”，系统展示感过强。
- 信息露出太干净，缺少误读、代价和制度缝隙。

### Focus Fields

- `scene_resistance`
- `information_carrier`
- `consequence_next_friction`

### B 版短改写

```text
复检告示贴出来那天，饭堂的粥又被兑了一瓢水。

苏衍端着碗坐在木屏风旁。屏风后面两个老杂役在分送柴牌，一个问后山那趟谁去，另一个压低声音：“别叫新来的，上回走到矿洞口，回来烧了三天。”

“什么矿洞。”有人咳了一声，“那地方以前不是挖矿的。”

“修炼室。”这两个字说完，筷子声立刻盖了上来。

苏衍没有马上起身。等看火杂役骂少人送柴，他才把空碗放下：“我去。”

看火杂役把一块熏黑的木牌丢给他：“天黑前回来。少一捆，明早替我刷锅。”

后山旧路旁有半截烂木牌，上面旧字写着“闭关旧室”，新刻的一行却是“矿洞废址，送柴止步”。同一个地方，两种名字。

矿洞里，苏衍看不懂墙上的符文。他反而松了口气。直到他看见符文旁的维护刻痕：回流，分配，筛。

最后一个字被划掉了一半。

他原以为灵气断裂是事故。事故不会被人一遍遍维护。

阵眼只剩拳头大小的金光。他碰到边缘时，没有立刻得到完整真相，只听见几段残缺汇报：“利用率下降……枯竭期……第二轮筛选……”

苏衍跪在冰冷石地上，先想起的不是阵法，而是饭堂那锅被兑稀的粥。

积分也是这样被分薄、回收、再分下去的。

他站起来时，袖里的送柴牌还带着油烟味。天黑前他得回去点名，明早还欠人一次刷锅。
```

### 剧情功能是否保留

保留。饭堂线索、矿洞旧修炼室、阵眼、枯竭不是事故、周期筛选和积分分配机制均保留。

### Dry Run 判断

压缩 packet 对信息节点有效。`information_carrier` 从老杂役、送柴牌、旧禁牌、维护刻痕、残缺汇报中连续生效，没有退化成路人闲聊一次性讲设定。

## 3. C5 群体公告 / 规则公布节点

### Baseline 摘要

元启在演武场竖古镜，宣布不走就默认保留积分并接受检测。镜面展示积分分配记录、古阵覆盖范围、苏衍积分曲线、古阵枯萎过程。元启说明积分来自古阵，三千年寿命、周期筛选和积分回收。众人哭骂，苏衍继续计算情绪碎片异常。

### Baseline 问题

- 信息由元启和镜面集中展示，像系统说明。
- 群体反应概括为哭骂，缺少个体选择压力。
- “没人走”是好节点，但代价没有充分落地。

### Focus Fields

- `scene_resistance`
- `information_carrier`
- `consequence_next_friction`

### B 版短改写

```text
元启把古镜竖起来时，演武场地上还留着昨日红签的蜡痕。

“想知道真相的，站到镜前来。不想知道的，可以走。”

人群动了一下。不是向前，是向后。

一个刚入门的杂役真转了半个身，旁边的人一把攥住他的袖口，低声说：“你走了，积分也不要了？”

元启像没看见。

“不走的人，默认保留积分。保留积分，就要接受检测。”

小杂役的袖口被攥出一团皱褶。没人走。苏衍也没有出声。

古镜亮起，镜面不是神迹，是账册。年份、姓名、积分起落一行行往下流。有人看见自己的名字，脖子先伸长，又缩回去。

山川水系铺开，金线从地下伸向山门、村镇、练功台和杂役宿舍。

“古阵。”元启说，“方圆三百里内，所有积分，都从这里分出去。”

镜面忽然跳出苏衍的曲线：四十五，一百一十二，六十一。

周围有人转头看他。柳青砚站在正式弟子那边，没有替他挡。

元启让镜面继续。金色古阵一层层变灰，名字一批批被划掉。

“每一次枯竭，都是筛选。能主动利用积分的人留下，只能被动接收的人，回收。”

有人骂了一句，声音刚出口就断了。镜面上他的名字亮了一下。

不是警告，是标记。

苏衍经络里的积分也热了一瞬。他看见那个想走的小杂役捂住胸口，袖口皱褶还没平。
```

### 剧情功能是否保留

保留。古镜、积分来源、古阵覆盖、苏衍曲线、三千年寿命、枯竭筛选、回收逻辑和检测标记均保留。

### Dry Run 判断

压缩 packet 对公告节点有效。公告没有被取消，但规则通过“不走的选择”、名字亮起、标记反应和苏衍曲线暴露来承担后果。

## 4. C4 情绪残留 / 情绪代价节点

### Baseline 摘要

苏衍离测试石只剩两步，把积分从一百一十二依次烧到九十、八十五、七十八、七十一、六十四。每个分数伴随情绪碎片或身体反应。按上测试石后三息回落，最终读数六十一。

### Baseline 问题

- 情绪像被消费的燃料条。
- 代价有身体反应，但不够影响选择和后续社交。
- 读者更容易记住分数下降，而不是苏衍承受了什么。

### Focus Fields

- `shame_or_avoidance`
- `consequence_next_friction`

### B 版短改写

```text
现在他离测试石只剩两步。

一百一十二。

这个数如果落在石头上，黑衣人会直接把他拖出杂役队列。

苏衍把手藏进袖子里，开始烧积分。

九十。胖少年的竞争欲碎开，那不是恨，是“我一定要被看见”的急。他的视线差点往正式弟子队列里找去。

苏衍把目光压回测试石。

八十五。白衣女弟子的焦虑炸开，他胸口一紧，差点退半步。后面的杂役低低骂了一声。

七十八。恨意上来，他咬住下唇。血腥味出来时，他差点抬手去擦。不能擦，白签桌就在旁边。

七十一。孤独感化开，像周围所有人都在说话，却没有一句是对他说的。他第二步偏了一寸，用脚尖硬把自己别回来。

六十四。

“下一个。”执事说。

苏衍把手按上测试石。

第一息，七十。第二息，六十五。第三息，六十一。

执事在竹简上划了一下。苏衍把手拿开，袖口擦过唇角，不知道血有没有干净。

他不会拿红签。

这个念头刚落下，柳青砚的声音从正式弟子队列里响起来。
```

### 剧情功能是否保留

保留。分数链、测试石三息窗口、最终读数六十一和后续柳青砚质疑均保留。

### Dry Run 判断

压缩 packet 对情绪节点仍有效，但这是 5 个片段中篇幅风险最高的一类。只选两个 focus fields 是正确的；若加入第三字段，容易把每个情绪碎片都展开成模板动作。

## 5. C4 章尾钩子压过人味节点

### Baseline 摘要

苏衍拿到白签后看柳青砚背影，想解释但无话可说。随后黑衣人抬头，天空裂缝出现，所有积分震颤，元启降临，宣布受古阵之命回收积分，枯竭提前、纯度不足者回收。夜里柳青砚远远看他后离开，苏衍总结认知优势的代价。

### Baseline 问题

- 关系余波有效，但很快被天幕裂缝和元启降临压过。
- 人物代价被放到尾段主题句里补写。
- 柳青砚夜里出现有效，但像补偿性人味。

### Focus Fields

- `private_want`
- `relationship_debt_change`
- `consequence_next_friction`

### B 版短改写

```text
苏衍拿到白签时，签角在掌心折出一道痕。

这不是通过，只是执事把问题往后推了一步。

队伍继续往前。苏衍站在枯松下，看着柳青砚回到正式弟子队列。他想过去说一句什么。谢谢。不关你的事。我有苦衷。每一句都像新的谎。

胖少年半抬了一下手，像想问自己刚才是不是惹了祸。苏衍没有回应。他现在回一个眼神，也会多欠一笔。

黑衣人的笔尖忽然停住。

苏衍抬头之前，先想起自己还没有叫住柳青砚。

晚了一息。

天空裂开了。

金色裂缝横在云边，所有人的积分同时震了一下。苏衍下意识去看柳青砚。她的手已经摸到剑柄，却没有看他。

执事攥紧竹简，嘴唇动了一下。

“三年。他说还要三年的。”

元启从裂缝中降下，掌心亮起圆形符文。

“受古阵之命，对青云宗进行积分回收。”

温暖从每个人经络里闪过，又立刻消失。不是检测，是标记。

“回收标准很简单。积分纯度。纯度高，保留。纯度不足——回收。”

柳青砚终于转头，目光掠过苏衍，又停在元启身上。

她没有再给他台阶。
```

### 剧情功能是否保留

保留。白签、关系余波、天幕裂缝、积分震颤、执事口型、元启降临、标记和回收标准均保留。

### Dry Run 判断

压缩 packet 对章尾钩子有效。B 版没有削弱大钩子，只把苏衍错过解释窗口嵌进钩子发生时，避免尾段再用主题句补人味。
