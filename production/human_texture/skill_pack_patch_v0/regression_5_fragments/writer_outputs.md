# Patched Writer Simulation Outputs

本文件模拟 `webnovel_writer` 在实验 patch 下接收 `human_texture` block 后的执行结果。

测试重点：

- 不逐项显性解释字段。
- 字段转成行为、误读、场景阻力、选择或后果。
- 保留原有网文推进、钩子、规则破局和节奏。
- 不靠风景、比喻、口语化或装饰性细节假装有人味。

## 1. C4 柳青砚关系节点

### Patched Writer Brief

```yaml
focus_fields:
  - shame_or_avoidance
  - relationship_debt_change
  - consequence_next_friction
visible_requirements:
  - "苏衍不能把撒谎写成纯理性隐藏。"
  - "柳青砚给台阶后不得立刻和解。"
  - "胖少年作证必须成为未还人情。"
avoid:
  - "不要写“这是关系债”。"
  - "不要用旁白总结信任破裂。"
```

### B 版短改写

```text
执事看向苏衍时，白签和红签还摆在小桌上。

苏衍张了张嘴，准备说“可能是你感知错了”。话到舌尖，他忽然觉得那句话太轻，也太脏。

“但是他输了。”

胖少年从正式弟子队列里挤出声音，脸涨得通红。他说完辩解，立刻低下头，像刚把自己的白签也押了出去。

苏衍没有看他。他知道自己欠了这句话，可现在点一下头，都会把人拖进来。

柳青砚往前两步，声音低到只有苏衍能听见。

“我是在给你台阶。你公开一点，就不用一个人扛。”

他停了一下。

“也不用把别人一起瞒进去。”

苏衍想到柳青砚掌心那道金线。他很想说“不关你的事”。那至少像是保护。

但他说出口的是：“我没有秘密。”

柳青砚退了一步。不是生气，是把刚递出来的东西收回去。

白签到手时，苏衍手心的汗把签角浸软了一点。
```

### Writer Regression Check

- 原剧情功能：保留。
- 字段显性化：未出现。
- 模板化风险：低。
- 篇幅风险：低。
- 人味来源：羞耻、回避、关系债、未还人情。

## 2. C3 饭堂 / 矿洞信息露出节点

### Patched Writer Brief

```yaml
focus_fields:
  - scene_resistance
  - information_carrier
  - consequence_next_friction
visible_requirements:
  - "主角不能直接读懂底层源码。"
  - "矿洞信息必须从饭堂秩序、旧牌、维护刻痕和残缺汇报露出。"
  - "进入后山必须留下误工或替班代价。"
avoid:
  - "不要让老杂役一次性讲完整设定。"
  - "不要堆无功能生活细节。"
```

### B 版短改写

```text
复检告示贴出来那天，饭堂的粥又被兑了一瓢水。

苏衍端着碗坐在木屏风旁。屏风后两个老杂役在分送柴牌，一个问后山那趟谁去，另一个压低声音：“别叫新来的，上回走到矿洞口，回来烧了三天。”

“什么矿洞。”有人咳了一声，“那地方以前不是挖矿的。”

“修炼室。”这两个字说完，筷子声立刻盖了上来。

苏衍没有马上起身。等看火杂役骂少人送柴，他才把空碗放下：“我去。”

看火杂役把熏黑的木牌丢给他：“天黑前回来。少一捆，明早替我刷锅。”

后山旧路旁有半截烂木牌，旧字写着“闭关旧室”，新刻的一行却是“矿洞废址，送柴止步”。

矿洞里，苏衍看不懂墙上的符文。直到他看见符文旁的维护刻痕：回流，分配，筛。

最后一个字被划掉了一半。

阵眼只剩拳头大小的金光。他碰到边缘时，没有立刻得到完整真相，只听见几段残缺汇报：“利用率下降……枯竭期……第二轮筛选……”

他站起来时，袖里的送柴牌还带着油烟味。天黑前他得回去点名，明早还欠人一次刷锅。
```

### Writer Regression Check

- 原剧情功能：保留。
- 字段显性化：未出现。
- 模板化风险：低。
- 篇幅风险：低到中；生活细节仍需控制密度。
- 人味来源：场景阻力、误读、信息载体、替班代价。

## 3. C5 群体公告 / 规则公布节点

### Patched Writer Brief

```yaml
focus_fields:
  - scene_resistance
  - information_carrier
  - consequence_next_friction
visible_requirements:
  - "公告不能取消，但必须变成强制选择现场。"
  - "没人走要有具体个体压力。"
  - "检测/标记要有即时后果。"
avoid:
  - "不要把规则完整改写成旁白说明。"
  - "不要用群体哭骂替代个体选择。"
```

### B 版短改写

```text
元启把古镜竖起来时，演武场地上还留着昨日红签的蜡痕。

“想知道真相的，站到镜前来。不想知道的，可以走。”

人群动了一下。不是向前，是向后。

一个刚入门的杂役真转了半个身，旁边的人一把攥住他的袖口，低声说：“你走了，积分也不要了？”

“不走的人，默认保留积分。保留积分，就要接受检测。”

小杂役的袖口被攥出一团皱褶。没人走。苏衍也没有出声。

古镜亮起，镜面不是神迹，是账册。年份、姓名、积分起落一行行往下流。

“古阵。”元启说，“方圆三百里内，所有积分，都从这里分出去。”

镜面忽然跳出苏衍的曲线：四十五，一百一十二，六十一。

周围有人转头看他。柳青砚站在正式弟子那边，没有替他挡。

金色古阵一层层变灰，名字一批批被划掉。

“每一次枯竭，都是筛选。能主动利用积分的人留下，只能被动接收的人，回收。”

有人骂了一句，声音刚出口就断了。镜面上他的名字亮了一下。

不是警告，是标记。
```

### Writer Regression Check

- 原剧情功能：保留。
- 字段显性化：未出现。
- 模板化风险：低。
- 篇幅风险：低。
- 人味来源：选择压力、群体目光、名字亮起、标记代价。

## 4. C4 情绪残留 / 情绪代价节点

### Patched Writer Brief

```yaml
focus_fields:
  - shame_or_avoidance
  - consequence_next_friction
visible_requirements:
  - "保留分数链。"
  - "每个情绪最多一个动作后果。"
  - "情绪残留要接到后续柳青砚质疑。"
avoid:
  - "不要把每个情绪展开成长段散文。"
  - "不要新增第三类情绪后果。"
```

### B 版短改写

```text
现在他离测试石只剩两步。

一百一十二。

苏衍把手藏进袖子里，开始烧积分。

九十。胖少年的竞争欲碎开，他的视线差点往正式弟子队列里找去。

八十五。白衣女弟子的焦虑炸开，他胸口一紧，差点退半步。

七十八。恨意上来，他咬住下唇。血腥味出来时，他差点抬手去擦。不能擦，白签桌就在旁边。

七十一。孤独感化开，他第二步偏了一寸，用脚尖硬把自己别回来。

六十四。

“下一个。”执事说。

苏衍把手按上测试石。

第一息，七十。第二息，六十五。第三息，六十一。

执事在竹简上划了一下。苏衍把手拿开，袖口擦过唇角。

他不会拿红签。

这个念头刚落下，柳青砚的声音从正式弟子队列里响起来。
```

### Writer Regression Check

- 原剧情功能：保留。
- 字段显性化：未出现。
- 模板化风险：中低；情绪动作可能重复，需要 Reviewer 盯住。
- 篇幅风险：中低；比 dry run 更压缩。
- 人味来源：遮掩、不属于自己的反应、下一场质问压力。

## 5. C4 章尾钩子压过人味节点

### Patched Writer Brief

```yaml
focus_fields:
  - private_want
  - relationship_debt_change
  - consequence_next_friction
visible_requirements:
  - "保留元启降临和回收标准。"
  - "关系余波必须嵌入钩子发生时。"
  - "不追加长尾主题句。"
avoid:
  - "不要削弱大钩子。"
  - "不要用夜里补一笔替代现场承受。"
```

### B 版短改写

```text
苏衍拿到白签时，签角在掌心折出一道痕。

这不是通过，只是执事把问题往后推了一步。

队伍继续往前。苏衍站在枯松下，看着柳青砚回到正式弟子队列。他想过去说一句什么。谢谢。不关你的事。我有苦衷。每一句都像新的谎。

胖少年半抬了一下手，像想问自己刚才是不是惹了祸。苏衍没有回应。他现在回一个眼神，也会多欠一笔。

黑衣人的笔尖忽然停住。

苏衍抬头之前，先想起自己还没有叫住柳青砚。

晚了一息。

天空裂开了。

金色裂缝横在云边，所有人的积分同时震了一下。苏衍下意识去看柳青砚。她的手已经摸到剑柄，却没有看他。

执事攥紧竹简，嘴唇动了一下：“三年。他说还要三年的。”

元启从裂缝中降下，掌心亮起圆形符文。

“受古阵之命，对青云宗进行积分回收。”

温暖从每个人经络里闪过，又立刻消失。不是检测，是标记。

“回收标准很简单。积分纯度。纯度高，保留。纯度不足——回收。”

柳青砚终于转头，目光掠过苏衍，又停在元启身上。

她没有再给他台阶。
```

### Writer Regression Check

- 原剧情功能：保留。
- 字段显性化：未出现。
- 模板化风险：低到中；“没有再给台阶”需避免后续反复使用。
- 篇幅风险：低。
- 人味来源：想解释但错过窗口、关系债被大钩子压住但未消失。
