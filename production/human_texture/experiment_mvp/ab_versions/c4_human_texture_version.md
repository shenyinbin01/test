# C4 Human Texture Version

说明：本片段为实验性 B 版，不替换正式正文。剧情功能保持：柳青砚指控，胖少年作证，证人规则压住指控，柳青砚给台阶，苏衍撒谎，关系破裂，苏衍拿到白签。

```text
执事的目光移到苏衍身上。那目光不像在审问，像在称重。

苏衍喉咙发紧。他知道执事昨晚那句“以后不要再去”是什么意思。执事知道积分会涨，也知道有人会因此消失。

但这里是演武场。

左边是正式弟子，右边是杂役。白签和红签摆在小桌上，签角朝外，像一排已经磨好的小刀。执事可以在矿洞口提醒他，却不能当着所有人的面替一个杂役遮掩。

苏衍张开嘴，准备说“可能是你感知错了”。

他说不出口。

那句话太轻，也太脏。柳青砚刚才看他的眼神不是要踩死他。她是在逼他把话说到明处。

“但是他输了。”

全场转头。

说话的是那个九十一分的胖少年。他站在正式弟子队列中间，胖脸涨得通红，一只手捏着自己的袖口，捏得布边皱成一团。

“他输给苏衍了。”胖少年指着柳青砚，声音先抖了一下，后面才硬起来，“输了以后感知可能不准。而且六十七和六十一，就这么点差距，谁说得清？”

他说完就低下头，像后悔自己把话扔出来了。

苏衍没看他。

他知道自己欠了这句话。也知道现在不能还。这个时候对胖少年点一下头，都可能把对方拖进更麻烦的地方。

柳青砚的嘴角扯了一下。不是因为胖少年拆他的台，是因为苏衍仍旧没有自己开口。

执事看了胖少年一眼。

“复检异常需要至少两个证人。”执事的声音很平，像在念一条谁都知道、谁都不敢碰的规矩，“提出指控的双方，各需一个同队证人。指控方来自正式弟子。被指控方来自杂役弟子。”

他转向杂役队列。

“杂役方有其他证人吗？”

苏衍看过去。

有个和他同屋的杂役把手往袖子里缩了缩。另一个人盯着地上的石缝，好像那里忽然长出了一株草。没人站出来。

苏衍不怪他们。

他甚至有点羡慕他们。只要不看他，不说话，他们今天还能拿白签，晚上还能回去抢一碗热一点的粥。

执事等了三息。

“没有证人。”

“等一下。”

柳青砚往前走了两步。

这次他没有看执事。他站在苏衍面前，比苏衍高半个头，袖口有一截被剑茧磨亮的布边。苏衍以前不会注意这种东西。现在他注意到了，因为那只手曾经露出过一道金色细纹。

“你以为我在害你？”

苏衍没有说话。

“我不是在害你。”柳青砚的声音压得很低，低到只有苏衍能听见，“我是在给你台阶。你如果在演武场上当着所有人公开一点，你就不用一个人扛。”

他停了一下。

“也不用把别人一起瞒进去。”

苏衍的指尖冷了一下。

柳青砚不知道矿洞。他也不知道苏衍昨晚已经从他掌心那道金线里看见了不该看的东西。

如果苏衍承认，他也许能把事情变成宗门程序里的异常。执事会记，黑衣人会查，柳青砚会被迫站到他旁边。

可柳青砚自己的秘密呢？

那道金线一旦被人问起，苏衍没有办法解释自己为什么知道。

他看着柳青砚，忽然很想说一句“不关你的事”。那至少像是保护。

但他说出口的是另一句。

“我没有秘密。”

柳青砚退了一步。

他没有愤怒。愤怒还可以吵，还可以追问。他只是把视线从苏衍脸上收了回去，像把一件刚递出去的东西拿回来。

苏衍这才发现自己的手心全是汗。白签还没拿到，他已经把掌心攥出了疼。

执事宣布继续复检。柳青砚回到正式弟子的队列里，没有再看他。

苏衍走过去拿签。

白色的。

签角被他的汗浸软了一点，贴在指腹上。苏衍把它攥住，没有抬头。

他通过了。

胖少年那句话、杂役队列的沉默、柳青砚退开的那一步，都留在他身后。
```
