# 第一章 归零

手机又震了。

林砚没看，他知道是什么。医院催缴短信，今天第三条。

“您有新的订单，请尽快送达。”

他扫了眼地址——老旧小区六楼，没电梯。膝盖已经开始疼了。

四点零三分。

他冲进单元门，电梯口贴着“维修中”。楼梯间昏暗，声控灯坏了一半。一步跨三级台阶，四楼转弯时膝盖一软，他扶住墙，喘了两口。

不能停。

平台超时罚款一单五块，他已经超了两单。

推开六楼防火门，走廊尽头一束斜阳照进来，灰尘在光线里飘。

然后他看见了。

走廊中间蹲着个老人，穿件洗得发白的旧夹克，正拿螺丝刀修电表箱。老人头顶——就在那束光线里——悬着一串淡金色的数字。

林砚眨了眨眼。

数字还在。

他揉了揉眼睛。那串数字没消失，像透过热浪看到的虚影，边缘泛着极淡的光。

六位数。开头是四。

他脑子里冒出一个念头：这什么鬼？

往前走了一步，想看清楚。数字突然模糊了一瞬，太阳穴像被针扎了一下，疼得他倒吸一口凉气。

老人抬头看他。

那眼神太平静了，像早知道他会站在那里。林砚心里一紧，赶紧移开视线，把外卖放在门口鞋柜上，转身就走。

下楼时脑子里全是那串数字。

六位数。四十多万？还是四百万？

一个穿破夹克、蹲走廊修电表的老人？

---

第二单送到市中心的高档公寓。

开门的男人四十岁上下，西装革履，手腕上的表在门廊灯下反光。家里装修气派，玄关摆着半人高的绿植。

林砚递餐盒时，男人正对着手机低声说话，语气压得很低：“我跟你说过多少次了，那笔钱周一必须到账……不是我不想给，是对方那边也有问题……”

林砚的视线掠过男人头顶。

又是一串数字。

但和老人不同——这串数字小得多，只有五位数，而且正在快速下跌。每眨一次眼，数字就掉几百。

林砚心跳快了一拍。

他脑子里迅速拼出一个答案：这些数字是身价，是财富。老人穿得破，但可能是隐形富豪；这个男人看起来光鲜，但正在破产。

男人挂断电话，脸色铁青，一把接过餐盒，砰地关上了门。

林砚站在门外，盯着紧闭的门。

那个数字太扎眼了。万一老人真有钱，今天就是唯一的机会。

手机震了一下。医院短信：“距离缴费截止时间还有四十五分钟。”

他看了眼订单列表，还有一单要送。手指在屏幕上悬了两秒，改了路线。

---

老旧小区门口的水果摊前，老人正弯腰挑苹果。

林砚把电动车停在路边，假装等人的样子，余光一直跟着老人。老人从一堆蔫苹果里挑出三个，和摊主磨了半天价，最后省了三毛六。

一个头顶挂着六位数的人，在水果摊前讨价还价两块钱？

林砚走过去，假装随意地问：“大爷，您住这小区啊？”

老人抬头看他，眼神和刚才一样平静：“有事？”

“没，没……”林砚卡壳了，“就是刚才看您修电表，挺厉害的。”

老人没接话，拎着三个苹果转身走了。

走了两步，突然停下来，回头看了林砚一眼。

“年轻人，眼神太好有时候不是好事。”

然后走了。

林砚站在原地，后背一阵发凉。那串数字还在老人头顶稳稳悬着，像是在嘲笑他的自作聪明。

手机震动。医院短信直接弹了全屏：“距离缴费截止时间还有三十分钟。逾期将暂停用药，请尽快补缴。”

四点三十二分。

他骑上电动车，油门拧到底。

---

市第三人民医院住院部的缴费窗口排着长队。

林砚冲进去的时候，额头的汗顺着脸颊往下淌。他掏出银行卡和手机，站在队伍末尾，手指在屏幕上反复计算余额。

三千一百块。

加上卡里的两千，一共五千一。还差三千一。

他翻了一遍通讯录。能借的人都借过了。上个月跟大学室友借的两千还没还，上周跟表哥借的一千五，对方已经暗示过好几次了。

队伍往前挪了两步。

林砚盯着缴费窗口上方的电子屏，脑子一片空白。他听见自己的心跳声，一下一下，敲在耳膜上。

轮到他的时候，他把银行卡递进去。

“余额不足。”窗口里的护士头也不抬，“差三千一，你回去筹够了再来。”

“能不能先交一部分……”他的声音干涩，“我爸的药不能停。”

护士终于抬头看了他一眼：“规定就是这样，欠费超过三天就要停药。你这不是第一天了，早该想办法。”

林砚张了张嘴，什么都没说出来。

他转身，往走廊深处走去。病房门半开着，他看见病床上父亲苍白的侧脸，呼吸机一下一下起伏。

然后他看见了。

父亲头顶——就在病房日光灯的冷白色光线里——浮现出一串淡金色的数字。

和老人一样，和那个客户一样。

但不一样的是，这串数字正在变淡。

不是下降，不是跳动。是像铅笔字被橡皮一层层擦掉，从边缘开始模糊、碎裂、消失。

五位数变成四位数。

四位数变成三位数。

三位数变成两位数。

林砚的呼吸停住了。他死死盯着那串数字，希望它停下来，希望它重新变清晰。可它没有。

数字边缘开始碎裂，像被风吹散的纸灰，一片一片消失在空气里。

最后一位数字闪烁了两下。

像灰烬里最后一点火星。

然后，彻底熄灭。

林砚的视野里，父亲头顶只剩一片空白。
