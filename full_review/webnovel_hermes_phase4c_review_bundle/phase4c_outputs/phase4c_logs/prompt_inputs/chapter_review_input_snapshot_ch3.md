# 第三章 第一次主动选择

走廊里的灯管嗡嗡响，像某种低沉的警告。

林砚站在重症监护室门外，手机屏幕已经黑了。那个女声说完“时间不多”就挂了，他再打过去，提示已关机。

他想透过门缝看里面的情况，但门关得太严，只看到护士的白色身影在玻璃后一闪而过。他往前迈了一步，手刚碰到门把手，一个护士就推门出来，差点撞上他。

“家属在外面等，里面正在抢救。”护士的语气很平淡，像在说今天天气不错。

“我爸他——”

“有情况会通知你。”

护士关上门，脚步声消失在走廊深处。监护仪的警报声从门缝里漏出来，时断时续，像某种不规则的心跳——也许就是父亲的心跳。林砚攥紧拳头，指甲掐进肉里，疼得他清醒了一点。

走廊里消毒水的味道让他想起母亲住院那一年。那时候他也是这样站在走廊里，什么都做不了。过了这么多年，他似乎有了什么“能力”，但还是什么都做不了。

他低头看自己的手。手指上还残留着送外卖时磨出的茧子，指节因为用力而泛白。他松开拳头，强迫自己深呼吸。

冷静。必须冷静。

他抬起头，目光扫过走廊。对面长椅上坐着一个中年妇女，穿着洗得发白的棉袄，手里攥着一张皱巴巴的缴费单。她的眼睛红肿，应该是刚哭过。林砚的目光无意间掠过她头顶——

那个数字。

他之前在急诊大厅见过类似的数字，但那时候他没仔细看。现在他盯着那个数字，发现它在动。不是剧烈跳动，而是一种缓慢的、几乎察觉不到的上升——从“37,200”慢慢跳到“37,500”，然后又停下来。

那个妇女看起来不像有钱人。她脚上的布鞋沾着泥，棉袄袖口磨得发亮。如果标签代表财富，她的数字不该是这个量级。

林砚皱起眉，继续盯着。

妇女的手机响了。她接起来，声音沙哑：“喂……嗯，我知道……我再想办法……”

她挂断电话，数字没变。她低头看着手里的缴费单，又抬头看了一眼监护室的门，然后掏出手机翻通讯录。她拨出一个号码，等了很久，没人接。数字往下掉了一截——“36,800”。

林砚心跳加速。

他看着她继续打电话。第三个电话通了，她低声说了几句，挂断后数字往上跳了一点——“37,000”。第四个电话没打通，数字又往下掉。

不是财富。是选择。

每一次她做出“继续治疗”的选择，数字就往上跳；每一次被拒绝，数字就停滞或下降。她的标签不是身价，而是她在为某个选择付出的代价。

监护室的门突然打开，护士探出头：“林砚家属？”

林砚猛地回过神：“在。”

“病人暂时稳定了，但还要观察。你先别走远。”

护士说完又关上门。林砚靠在墙上，长长地吐出一口气。手指还在发抖，但他强迫自己把注意力拉回那个妇女身上。

她已经站起来，朝缴费窗口走去。林砚跟了几步，保持距离，看着她站在窗口前犹豫。她手里攥着一沓钱，厚薄大概能看出来——不够。

窗口里的收费员探出头：“大姐，两个方案你都看了，便宜的做基础治疗，贵的加靶向药，效果差不少。你选哪个？”

妇女咬着嘴唇，手里的钱攥得更紧了。她低头看了一会儿，说：“便宜的。”

林砚盯着她头顶的数字。没有变化。

他脑子里闪过一个念头——如果她选了贵的呢？

他不知道这个念头从哪来的，也不知道自己为什么会在意一个陌生人的选择。但他已经走上前，说：“阿姨，那个贵的方案，我听说效果真的不一样。”

妇女抬头看他，苦笑了一下：“没钱啊。”

林砚看着她手里的钱，又看了一眼她的数字——还是没变。他犹豫了两秒，伸手从口袋里掏出今天结算的外卖费——两千块，皱巴巴的，还没来得及存进卡里。

“先借你，不急还。”

妇女愣住了，张了张嘴，没说出话。林砚把钱塞进她手里，眼睛盯着她头顶的数字。

数字边缘亮了一下，像被什么力量推了一把，然后开始上升——从“37,000”跳到“39,800”，又跳到“41,200”。

林砚盯着那个数字，心跳快得像擂鼓。

妇女眼眶红了，声音发颤：“小伙子，你——”

“没事，你先缴费。”

妇女转过身，把钱递进窗口：“我选贵的。”

林砚看着她头顶的数字继续上升，直到稳定在“43,500”。他脑子里一片混乱——两千块换来了标签的变动，但这意味着什么？他还没想明白，一阵剧烈的头痛突然袭来，像有人拿钉子从太阳穴扎进去。

眼前的走廊开始模糊，耳鸣声嗡嗡响，盖过了周围所有的声音。

他踉跄着后退，撞在墙上，蹲下来抱住头。视野里的颜色开始褪去，变成灰白一片。他用力揉太阳穴，指甲掐进皮肤，疼痛让他勉强保持清醒。

不知道过了多久，耳鸣声慢慢消退，视力一点一点恢复。

他蹲在医院一楼大厅角落的长椅上，大口喘气。手机震动了一下，是平台发来的派单提醒——他还有三单没送，已经超时了。

他看了眼时间，下午四点十二分。从父亲被送进抢救室到现在，只过了不到两个小时，但他感觉像过了一整天。

他站起来，腿有点发软。手里还攥着手机，屏幕上弹出扣分通知——超时配送，信用分扣5分，罚款50元。

他关掉通知，靠在墙上，闭着眼睛喘气。不知道值不值。两千块，加上超时扣分罚款，加上刚才那一阵头痛——他付出了这么多，只是为了让一个陌生人的标签往上跳了几千块。

但他看到那个妇女走进监护室时，脸上多了一点希望。她回头看了他一眼，嘴唇动了动，像是说了句谢谢。

林砚没听清，也没追问。

他掏出手机，给平台发了个请假申请——今天是送不了了。系统立刻弹出确认通知，附带一句提醒：“您的信用分已低于90分，请尽快完成订单恢复信用。”

他关掉手机，往一楼大厅走。他想去缴费窗口看看父亲的费用，刚走到大厅中央，路过一面落地镜。

他无意间扫了一眼镜子里的自己——头顶那个数字，跳了一下。

林砚停下脚步。

他记得之前在医院门口看过自己的标签——那时候是一个普通的五位数，和周围人差不多。但现在，那个数字变了。不是变高，也不是变低，而是数字本身在闪烁，像接触不良的灯泡，忽明忽暗。

他走到旁边一个路过的病人身边——那人头顶的数字稳定清晰，“12,800”。又走到一个护士身边——“45,600”，也是稳定的。

只有他。

他回到镜子前，盯着自己头顶的数字。那个数字确实在闪烁，边缘还有一层淡淡的、几乎看不见的暗红色光晕。他伸手去摸，手指穿过空气，什么也没碰到。

他盯着镜子里的自己，那个数字又跳了一下——从“8,200”跳到“8,100”，然后停住，又开始闪烁。

林砚的呼吸变得急促。他低头看自己的手，又抬头看镜子。别人的标签是稳定的，只有他的在闪。这意味着什么？他的标签和别人的不一样？还是他自己的选择正在改变什么？

他想再看清楚一点，但头痛又开始隐隐发作，太阳穴像被针扎。

他闭上眼睛，深呼吸，强迫自己冷静下来。等他再睁开眼，那个数字还在闪烁，但暗红色的光晕已经淡了一些。

林砚站在镜子前，盯着自己头顶那个不稳定的数字，脑子里只有一个念头——

他的标签，和所有人都不一样。
