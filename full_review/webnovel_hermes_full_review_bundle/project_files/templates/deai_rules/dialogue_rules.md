# 对白规则

## 口吻差异化

每个角色的对话必须有至少 2 个可辨识特征：
- 句式偏好（短句/长句/反问/设问）
- 用词习惯（特定口头禅、特定表达）
- 语气节奏（停顿位置、语速暗示）
- 信息透露方式（直接/迂回）

同一个场景中，如果两个角色的对话摘掉角色名后看不出区别，属于失败。

## 潜台词

角色对话的必备层次：
- 表面意思（说出口的话）
- 真实意图（角色真正想表达的东西）
- 隐藏信息（角色不想让对方知道的东西）

每段对白至少体现后两者之一。

## 避免"念台词"

以下情况的对话必须修改：
- 角色解释读者已经知道的事实
- 角色解释自己正在做的事（"我正在打开这扇门"）
- 一个角色重复另一个角色刚说过的话来强调
- 对话里出现了现实中不会有人说的话（"妈，你知道的，这是我们家欠医院的第三笔钱"）

## 对话标记

- 不要频繁使用"他说""她说"
- 用动作替代：他点了根烟："然后呢？"
- 在紧张场景中可以使用无标记对话（纯对话流）
- 每个对话标记之间不超过 3 轮对话

## 情绪传递

对话的情绪层次通过以下方式传递（按优先顺序）：
1. 说话内容本身（最优先）
2. 伴随动作/身体反应
3. 说话节奏（标点符号/句子长度）
4. 用词选择
5. 语气标记（极少使用，只用在不言自明无法传递的场合）
